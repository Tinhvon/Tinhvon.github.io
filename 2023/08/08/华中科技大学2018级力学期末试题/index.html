选择题

( )1.万有引力常量\(G\)的量纲为

\(A.MT^{-2}\) \(B.LMT^{-2}\) \(C.L^{-1}MT^{-2}\) \(D.L^3M^{-1}T^{-2}\)

( )2.一球从\(5m\)高处自由下落至水平桌面上,反跳至\(3.2m\)高处,所经历的总时间为\(1.9s\),则该球与桌面碰撞期间的平均加速为（设重力加速度\(g= 10m/s^2\))

\(A.\)大小为\(180m/s^2\),方向竖直向上

\(B.\)大小为\(180m/s^2\),方向竖直向下

\(C.\)大小为\(20/s^2\),方向竖直向上

\(D.\)零

( ) 3.一只质量为\(m\)的猴子,原来抓住一根用绳吊在天花板上的质量为\(M\)的直杆,悬线突然断开,小猴则沿杆子竖直向上爬以保持他离地面高度不变,此时直杆下落的加速度为

\((A) g\) \((B) mg/M\) \((C) (M + m)g/M\) \((D) (M + m)g/(M − m)\) \((E) (M − m)g/M\)

( ) 4.以下列4种方式将质量为\(m\)的物体提高\(10m\),提升力做功最小的是

\((A)\)将物体由静止开始匀加速提升\(10m\),使速度达到\(5m/s\)

\((B)\)物体从初速度\(10m/s\)匀减速上升\(10m\),使速度减到\(5m/s\)

\((C)\)以\(5m/s\)的速度匀速提升

\((D)\)以\(10m/s\)的速度匀速提升

( ) 5.潮汐摩擦将导致

\((A)\)地球自转越来越快,月球离地球越来越近

\((B)\)地球自转越来越快,月球离地球越来越远

\((C)\)地球自转越来越慢,月球离地球越来越近

\((D)\)地球自转越来越慢,月球离地球越来越远

( ) 6.花样滑冰运动员绕通过自身的垂直轴转动,开始时两臂伸开,转动惯量为\(I_0\),角速度为\(ω_0\).然后她将两臂收回,使转动惯量减少为\(I_0/2\),则此时她转动的角速度为:

\((A)\sqrt{ω_0}/2\) $(B)ω_0/ 2 $ \((C) 2ω_0\) \((D)2ω_0\)

( ) 7.当空气通过钢管在两悬挂等高的轻球之间喷出时:

\((A)\)由伯努利方程可证两球将分开

\((B)\)由伯努利方程可证两球将靠拢

\((C)\)由牛顿定律可证两球将分开

\((D)\)由牛顿定律可证两球将不动

( ) 8.简谐振动的频率为\(f\) ,则动能变化的频率为

\((A)f/2\) \((B) 2f\) \((C)f\) \((D)2f\)

( ) 9.在媒质中传播着一列平面简谐波,各质点均作简谐振动,任取一质点,在振动的过程中,其动能为Ek,势能为Ep,总机械能为E,则有

\((A)E = E_k + E_p\)为一恒量

\((B)E_k\)增大时,\(E_p\)减少;\(E_k\)减少时,\(E_p\)增大

\((C)\)\(E_k\)和\(E_p\)同时增大,同时减少,但\(E_k = E_p\)

\((D)\)\(E_k\)和\(E_p\)同时增大,同时减少,且总有\(E_k = E_p\)

( ) 10.关于光速不变原理,下列说法错误的是:

\((A)\)真空中的光速与光源运动无关.

\((B)\)真空中的光速与观测者的运动无关.

\((C)\)真空中的光速与光的颜色无关.

\((D)\)可通过测量不同情况下的单向光速来验证其正确性.

是水平方向.则子弹在穿透过程中所受的冲量的大小为_____ ,方向为_____.

第三次拉伸弹簧与第二次拉伸弹簧弹力所做功之比为_____.

恒”或“不守恒”）.

面垂直的轴的转动惯量为_____.
