Chapter5-02定积分的性质

定积分

定积分的性质

线性性

Theorem:定积分运算满足线性性质,即:

\(\int_{a}^{b}[\alpha f(x)+\beta g(x)]\mathrm{d}x = \alpha \int_{a}^{b}f(x)\mathrm{d}x + \beta \int_{a}^{b}g(x)\mathrm{d}x\).

可加性

Theorem:定积分满足可加性,即:

\(\int_{a}^{b}f(x)\mathrm{d}x=\int_{a}^{c}f(x)\mathrm{d}x+\int_{c}^{b}f(x)\mathrm{d}x\).

只要满足可积性,\(a,b,c\)可以为任何实数.

保序性

Theorem:若\(f(x)\leq g(x),x\in[a,b]\),则\(\int_{a}^{b}f(x)\mathrm{d}x \leq \int_{a}^{b}g(x)\mathrm{d}x\).

据此我们可以得到一个推论:

Theorem:若\(f(x)\geq 0\),则\(\int_{a}^{b}f(x)\mathrm{d}x \geq 0\).

绝对可积性

Theorem:\(|f(x)|\)在\([a,b]\)可积且\(|\int_{a}^{b}f(x)\mathrm{d}x| \leq \int_{a}^{b}|f(x)|\mathrm{d}x\)

估值定理

Theorem:若\(m \leq f(x) \leq M\),则\(m(b-a) \leq \int_{a}^{b}f(x)\mathrm{d}x \leq M(b-a)\).

第一积分中值定理

Theorem:若\(f(x)\in C[a,b]\),\(g(x)\)在\([a,b]\)上可积不变号,则\(\exists \xi \in [a,b]\),使得\(\int_{a}^{b}f(x)g(x)\mathrm{d}x =f(\xi)\int_{a}^{b}g(x)\mathrm{d}x\).

当\(g(x) = 1\),则\(\exists \xi \in [a,b]\),使得\(\int_{a}^{b}f(x)\mathrm{d}x =f(\xi)(b-a)\).

第二积分中值定理

Theorem:若\(f(x)\)在\([a,b]\)单调,则\(\exists \xi \in [a,b]\)使得\(\int_{a}^{b}f(x)g(x)\mathrm{d}x =f(a)\int_{a}^{\xi}g(x)\mathrm{d}x +f(b)\int_{\xi}^{b}g(x)\mathrm{d}x\) .

柯西-施瓦兹不等式

Theorem:若\(a<b\),则:

\((\int_{a}^{b}f(x)g(x)\mathrm{d}x)^2 \leq \int_{a}^{b}f^2(x)\mathrm{d}x\int_{a}^{b}g^2(x)\mathrm{d}x\)

证明如下:

Proof

易知:\([f(x)+\lambda g(x)] \geqslant 0 \quad \lambda \in R \\\)

则\(\Rightarrow g^2(x) \lambda^2+2 f(x) g(x) \lambda+f^2(x) \geqslant 0 \\\) 两边积分:\(\int_a^b g^2(x) d x \cdot \lambda^2+2 \int_a^b f(x) g(x) d x \cdot \lambda+\int_a^b f^2(x) d x \geqslant 0 \\\)

故有:\(\Delta=\left(2 \int_a^b f(x) g(x) d x\right)^2-4 \int_a^{b} f^2(x) d x \int_a^b g^2(x) d x \leq 0 \\\) 由此可证:\(\Rightarrow (\int_a^b f(x) g(x) d x)^2 \leq \int_a^b f^2(x) d x \int_a^b g^2(x) d x\)

例题

最后给出一些例题吧:

Example 1:计算\(\int_{-2}^4 \mathrm{sgn} x\)

Solution \(I = \int_{-2}^0(-1)\mathrm{d}x+\int_{0}^4(1)\mathrm{d}x = 2\)

Example 2:求\(f(x)=x^2\)在\([0,1]\)上的平均值\(I\).

Solution \(I = \cfrac{1}{1-0}\int_0^1x^2\mathrm{d}x = \cfrac{1}{3}\)

Example 3:比较大小\(I_1=\int_0^1\cfrac{x}{1+x}\mathrm{d}x,I_2 = \int_0^1\ln(1+x)\mathrm{d}x\).

Solution 易知\(\cfrac{x}{1+x} \leq \ln(1+x) \Rightarrow I_1 \leq I_2\)

Example 4:估值\(I = \int_{\pi/4}^{\pi/2}\cfrac{\sin{x}}{x}\mathrm{d}x\)

Solution \(\cfrac{2}{\pi}\leq \cfrac{\sin{x}}{x}\leq \cfrac{2\sqrt{2}}{\pi} \Rightarrow \cfrac{1}{2} < I < \cfrac{\sqrt{2}}{2}\)
