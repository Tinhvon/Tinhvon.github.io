不定积分

不定积分的概念

定义

首先我们给出原函数与不定积分的定义:

Definition:设f(x),F(x)f(x),F(x)f(x),F(x)在区间III上有定义,若F′(x)=f(x),x∈IF'(x)=f(x),x\in IF′(x)=f(x),x∈I,则称F(x)F(x)F(x)为f(x)f(x)f(x)的一个原函数.

Definition:f(x)f(x)f(x)在III上全体原函数称为f(x)f(x)f(x)在III上的不定积分,记作∫f(x)dx\int f(x) \mathrm{d}x∫f(x)dx

在给出这些定义后,我们做一些补充.

Note 1:若F(x)F(x)F(x)是f(x)f(x)f(x)的一个原函数,则F(x)+CF(x)+CF(x)+C也是f(x)f(x)f(x)的原函数,且任意两个原函数之间相差一个常数.即:

∫f(x)dx=F(x)+C(F′(x)=f(x))\int{f(x)}\mathrm{d}x=F(x)+C(F'(x)=f(x)) ∫f(x)dx=F(x)+C(F′(x)=f(x))

Note 2:连续函数一定存在原函数.

Note 3:积分与微分在相差一个常数的意义下,互为逆运算.即:

∫f′(x)dx=f(x)+C⋅∫df(x)=f(x)+Cddx∫f(x)dx=f(x)\int{f'(x)}\mathrm{d}x = f(x)+C\cdot\int{\mathrm{d}f(x)}=f(x)+C \\ \cfrac{\mathrm{d}}{\mathrm{d}x}\int{f(x)}\mathrm{d}x = f(x) ∫f′(x)dx=f(x)+C⋅∫df(x)=f(x)+Cdxd​∫f(x)dx=f(x)

Note 4:积分运算满足线性关系,即:

∫[αf(x)+βg(x)]dx=α∫f(x)dx+β∫g(x)dx\int{[\alpha f(x)+\beta g(x)]}\mathrm{d}x = \alpha \int{f(x)}\mathrm{d}x+\beta \int{g(x)}\mathrm{d}x ∫[αf(x)+βg(x)]dx=α∫f(x)dx+β∫g(x)dx

Note 5:积分常数的确定方法:先求出原函数,再根据条件确定常数CCC.

针对Note 5给出一个例题:

Example 1:已知曲线上任意一点切线斜率为该点横坐标的2倍,且曲线过(1,2)(1,2)(1,2),求曲线方程.

Solution:根据题意有dydx=2x⇒y=x2+C\cfrac{\mathrm{d}y}{\mathrm{d}x} = 2x \Rightarrow y = x^2+Cdxdy​=2x⇒y=x2+C

代入有2=1+C⇒C=12=1+C \Rightarrow C = 12=1+C⇒C=1

故曲线方程为C:y=x2+1C:y=x^2+1C:y=x2+1

Note 6不定积分的几何意义:若F(x)F(x)F(x)是f(x)f(x)f(x)的一个原函数,则称y=F(x)y=F(x)y=F(x)是f(x)f(x)f(x)的一条积分曲线.∫f(x)dx\int{f(x)}\mathrm{d}x∫f(x)dx为y=F(x)y=F(x)y=F(x)沿纵轴任意平移所得的积分曲线族.

基本积分公式

得到不定积分的概念后,我们给出基本函数的积分公式,证明略:

**1 **: ∫0dx=C\int{0}\mathrm{d}x = C∫0dx=C

**2 **: ∫xαdx=xα+1α+1+C\int{x^\alpha}\mathrm{d}x = \cfrac{x^{\alpha+1}}{\alpha+1}+C∫xαdx=α+1xα+1​+C

**3 **:∫dxx=ln⁡∣x∣+C\int{\cfrac{\mathrm{d}x}{x}} = \ln{|x|}+C∫xdx​=ln∣x∣+C

**4 **:∫axdx=axln⁡a+C,∫exdx=ex+C\int{a^x}\mathrm{d}x = \cfrac{a^x}{\ln a}+C,\int{e^x}\mathrm{d}x = e^x+C∫axdx=lnaax​+C,∫exdx=ex+C

**5 **:∫sin⁡xdx=−cos⁡x+C\int{\sin{x}}\mathrm{d}x = -\cos{x}+C∫sinxdx=−cosx+C

**6 **:∫cos⁡xdx=sin⁡x+C\int{\cos{x}}\mathrm{d}x=\sin{x}+C∫cosxdx=sinx+C

**7 **:∫1cos⁡2xdx=tan⁡x+C\int{\cfrac{1}{\cos^2{x}}}\mathrm{d}x = \tan{x} + C∫cos2x1​dx=tanx+C

**8 **:∫1sin⁡2xdx=−cot⁡x+C\int{\cfrac{1}{\sin^2{x}}}\mathrm{d}x = -\cot{x} + C∫sin2x1​dx=−cotx+C

**9 **:∫dx1−x2=arcsin⁡x+C\int{\cfrac{\mathrm{d}x}{\sqrt{1-x^2}}} = \arcsin{x} + C∫1−x2​dx​=arcsinx+C

**10 **:∫dx1+x2=arctan⁡x+C\int{\cfrac{\mathrm{d}x}{1+x^2}} = \arctan{x}+C∫1+x2dx​=arctanx+C
