Chapter4-03积分类型

不定积分

积分类型

在了解基本的不定积分积分方法后,我们已经可以求出大部分函数的不定积分.下面我们就来利用这些方法求解几种类型函数的不定积分.

最简积分

求解这些函数的不定积分,均可通过对其进行代数变形,换元等方式求出.直接给出例题:

Example 1:计算$\int \cfrac{d x}{1+e^x}$

Solution

$\begin{aligned}I &=\int \frac{1+e^x-e^x}{1+e^x} d x \\
& =x-\ln \left(1+e^x\right)+C
\end{aligned}$

Example 2:$\int{x^x(1+\ln{x})}\mathrm{d}x = x^x + C$.求解过程略

Example 3:计算$\int \cfrac{\ln \left(x+\sqrt{1+x^2}\right)}{\sqrt{1+x^2}} \mathrm{d} x$

Solution

$\begin{aligned} I& =\cfrac{1}{2} \ln \left(x+\sqrt{1+x^2}\right)+C \\
& =\int \ln \left(x+\sqrt{1+x^2}\right) \mathrm{d}\ln \left(x+\sqrt{1+x^2}\right)
\end{aligned}$

有理函数的积分

首先给出有理函数的定义:

Definition:形如$R(x) = \cfrac{b_mx^m+b_{m-1}x^{m-1}+…+b_1x+b_0}{a_nx^n+a_{n-1}x^{n-1}+…+a_1x+a_0} = \cfrac{P(x)}{Q(x)}$这样的函数被称为有理函数.一般来讲,当$m \geq n$时,为假分式,反之为真分式.

$R(x)$可以写成多项式与真分式的和.

给出有理函数的定义后,我们不加证明地给出多项式分解定理,内容如下:

Theorem:任何实函数多项式$Q(x)$总可以唯一分解成若干一次因式与二次因式之积.即:

有理函数的积分基于部分分式定理,其内容如下:

Theorem:真分式$\cfrac{P(x)}{Q(x)}$可唯一分解为如下形式:

$\cfrac{P(x)}{Q(x)}=\left[\cfrac{A_{11}}{x-\alpha_1}+\cfrac{A_{12}}{\left(x-\alpha_1\right)^2} \cdots \cdots+\cfrac{A_{1 r_1}}{\left(x-\alpha_1\right)^{r_1}}\right]+…+\left[\cfrac{A_{k 1}}{x-\alpha_k}+\cfrac{A_k}{\left(x-\alpha_k\right)^2}+\cdots+\cfrac{A_{k r_k}}{\left(x-\alpha_k\right)^{r_k}}\right]+…+ \\
+\left[\frac{B_{11} x+C_{11}}{x^2+p_1 x+q_1}+\cdots \cdots+\frac{B_{1s_1} x+C_{1s_1}}{\left(x^2+p_1 x+g_1\right)^{s_1}}\right] + …\left[\frac{B_{l1} x+C_{l1}}{x^2+p_l x+q_l}+\cdots \cdots+\frac{B_{ls_l} x+C_{ls_l}}{\left(x^2+p_l x+q_l\right)^{s_l}}\right]
$

在以上定理基础下,有理函数的积分可以拆解为以下积分的线性组合:

Integral 1: $\int{p(x)}\mathrm{d}x$

Integral 2: $\int{\cfrac{\mathrm{d}x}{x-a}}$

Integral 3: $\int{\cfrac{\mathrm{d}x}{(x-a)^n}}(n \ne 1)$

Integral 4: $\int{\cfrac{Mx+N}{x^2+px+q}}\mathrm{d}x$

Integral 5: $\int{\cfrac{Mx+N}{(x^2+px+q)^n}}\mathrm{d}x$

关于Integral 5其求解方法如下:

Solution

$\begin{aligned}I &=\cfrac{M}{2} \int \cfrac{2 x+p}{\left(x^2+p x^2+q\right)^n} \mathrm{d}x
+( N - \cfrac {Mp}{2} ) \int \cfrac {\mathrm{d}x } { ( x ^ { 2 } + p x + q ) ^ { n } } \\ \end{aligned}$

现在我们求解$\int \cfrac{d x}{\left(x^2+p x+q\right)^n}$即可,通过代换我们可以化为$\int \cfrac{d x}{\left(x^2+a^2\right)^n}=J_n$
易知$J_1= \cfrac{1}{a}\arctan{\cfrac{x}{a}}+C$

利用分部积分法有:

$\begin{aligned}
J_n & =\frac{x}{\left(x^2+a^2\right)^n}-\int x d \frac{1}{\left(a^2+a^2\right)^n} \\
&=\frac{x}{\left(x^2+a^2\right)^n}+2 n \int \frac{x^2+a^2-a^2}{\left(x^2+a^2\right)^{n+1}} d x \\
& =\frac{x}{(x^2+a^2)^n)}+2 n J_n-2 n a^2 J_{n+1} \\
&\Rightarrow J_{n+1}=\cfrac{2 n-1}{2na^2} J n+\cfrac{x}{\left.2 n a^2(x^2+a^2\right)^n}
\end{aligned}$

利用递推关系即可求得.

求出Integral 1到Integral 5后,我们便可求出所有的有理函数积分.下面给出几个例题:

Example 1:计算$\cfrac{\mathrm{d}x}{1+x^3}$

Solution

$\begin{aligned} I &=\int \cfrac{\mathrm{d}x}{(x+1)\left(x^2-x+1\right)} \\
& =\int \cfrac{\frac{1}{3}}{1+x} \mathrm{d}x -\int \frac{x-2}{3\left(x^2-x+1\right)} \mathrm{d}x \\
& =\frac{1}{3} \ln |1+x|-\frac{1}{6}|\ln | x^2-x+1 \mid+\frac{1}{\sqrt{3}} \tan ^{-1} \frac{2 x-1}{\sqrt{3}}+C
\end{aligned}$

Example 2:计算$\int \cfrac{2 x^5+6 x^3+1}{x^4+3 x^2} d x

Solution

$\begin{aligned} I &=\int 2 x d x
+\cfrac{1}{3} \int \cfrac{x^2+3-x^2}{x^2\left(x^2+3\right)} d x \\ &=x^2-\cfrac{1}{3 x} -\cfrac{1}{3 \sqrt{3}} \tan ^{-1} \cfrac{x}{\sqrt{3}}+C
\end{aligned}$

三角函数有理式积分

当被积函数可以写为三角函数形式时,即$\int{R(\sin{x},\cos{x})}\mathrm{d}x$时,其有以下通法:

  • 记$t = \tan{\cfrac{x}{2}} \Rightarrow \sin{x} = \cfrac{2t}{1+t^2},\cos{x} = \cfrac{1-t^2}{1+t^2},\mathrm{d}x = \cfrac{2\mathrm{d}t}{1+t^2}$
  • 原积分化为$\int{R(\cfrac{2t}{1+t^2},\cfrac{1-t^2}{1+t^2})}\cfrac{2\mathrm{d}t}{1+t^2}$,此时转化为有理函数的积分,此时可解.

下面给出几个例题:

Example 1:计算$\int \cfrac{\cos ^4 x}{\sin ^3 x} d x$

Solution

$\begin{aligned} I &=\int \frac{\cos ^4 x}{\sin ^4 x} \mathrm{d}\cos x \\
& =\int \frac{-\cos ^4 x}{\left(1-\cos ^2 x\right)^2} \mathrm{d}\cos x \quad \cos x=t \end{aligned}$

Note:当被积函数满足$R(-\sin{x},\cos{x}) = -R(\sin{x},\cos{x})$时,直接换元$t = \cos{x}$即可.

Example 2:计算$\int \cfrac{\cos ^3 x}{\sin ^4 x} d x$

Solution

$I = \int \cfrac{(1-\sin^2x)}{\sin ^4 x} d({\sin{x}})$

Note:当被积函数满足$R(\sin{x},-\cos{x}) = -R(\sin{x},\cos{x})$时,直接换元$t = \sin{x}$即可.

Example 3:计算$\int \cfrac{d x}{\sin ^4 x \cos ^2 x}$

Solution

$\begin{aligned} I &=\int \frac{d \tan x}{\left(1-\cos ^2 x\right)^2} \\
& =\int \frac{d \tan x}{\left(1-\frac{1}{1+\tan ^2 x}\right)^2} ,\tan x=t \\
&
\end{aligned}$

Note:当被积函数满足$R(-\sin{x},-\cos{x}) = -R(\sin{x},\cos{x})$时,直接换元$t = \tan{x}$即可.

无理根式积分

  • 对于$\int{R(x,\sqrt[m_1]{(\cfrac{ax+b}{cx+d})^{n_1}},…,\sqrt[m_r]{(\cfrac{ax+b}{cx+d})^{n_r}})}\mathrm{d}x$类型的积分,我们直接换元$t = \sqrt[m]{\cfrac{ax+b}{cx+d}}$,其中$m$是$m_i$的最小公倍数
  • 对于$\int{R(x,\sqrt{ax^2+bx+c})}\mathrm{d}x$类的积分我们直接利用Euler变换,如下:

$a > 0$,令$\sqrt{ax^2+bx+c}= t \pm \sqrt{a}x$

$c > 0 $,令$\sqrt{ax^2+bx+c}= xt \pm \sqrt{c}$

$ax^2+bx+c$有两个不等实根$\lambda ,\mu$时,$\sqrt{ax^2+bx+c} = t(x-\lambda)$.
