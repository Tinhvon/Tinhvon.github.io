光学-Chapter2-光的传播

光的传播

光的传播及光与物质相互作用中的基本问题,可概括为以下方面：

透射：入射光经过折射,穿过物体后的出射.

反射：当光在两种介质分界面上改变传播方向时,又返回原来介质中的现象.

折射：光从一种透明介质进入另一种透明介质时,传播方向发生变化的现象.

散射：光通过不均匀介质时,一部分光偏离原方向传播的现象.

光与物质每一次相遇,可以看成一个协同事件,此时光子会与真空中的原子产生作用,产生散射,实际上,光的传播的核心问题是有关散射的问题.不论透射、折射、反射,都是散射的宏观体现.

散射、反射、折射

散射

散射可以分为几类： 瑞利散射：入射光的波长远大于物体尺寸,发生的散射,其强度正比于\(\frac{1}{\lambda^4}\).

米氏散射：入射光的波长与物体尺寸相当时,发生的散射,其强度正比于\(\frac{1}{\lambda^2}\).

拉曼散射：由于组成物质的分子,原子振动与转动产生的散射.

布里渊散射：由于物质密度的涨落产生的散射.

主要讨论瑞利散射,强度与入射光的波长的四次方成反比,由电偶极辐射的光强分布公式： \[ I(\theta) = \cfrac{\mu_0^2\omega^4}{32\pi^2c^3\varepsilon_0}\cfrac{\sin^2 \theta}{r^2} \] 我们可知,各个方向散射光的强度是不一样的.

惠更斯原理

在讨论惠更斯原理之前,首先引入两个概念：波面和波线

波面：振动状态相同的质点组成的面.

波线：与波面垂直的线,表示波的传播方向.

惠更斯原理(Huygens Principle)：介质中任意波面上的各点,都可以看作是发射子波的波源,此后任意时刻,这些子波前进方向的包络面是新的波面.

利用Huygens原理,我们可以解释对应的光学现象：

光的直线传播：对于已知的波面,通过作图的方法可以确认下一时刻的平面波、球面波等波面,利用两者垂直关系,可以确定波的传播方向.

光的衍射：光到达狭缝时,缝上各点都可看作子波源,作出子波包络,得到新的波面.在缝的边缘处,波的传播方向发生改变.(注意：Huygens原理只能解释波的传播方向,不能解释波的强度,所以无法说明衍射现象与障碍物的尺寸关系.)

反射

反射可以分为镜面反射与漫反射,他们的定义分别如下：

镜面反射(Spetecular reflection):若反射面比较光滑,当平行入射的光线射到这个反射面时,仍会平行地向一个方向反射出来,这种反射就属于镜面反射.

漫反射(Diffuse reflection)：投射在粗糙表面上的光向各个方向反射的现象.当一束平行的入射光线射到粗糙的表面时,表面会把光线向着四面八方反射,所以入射光线虽然互相平行,由于各点的法线方向不一致,造成反射光线向不同的方向无规则地反射,这种反射称之为“漫反射”或“漫射”.

反射的基本规律：

入射光线、反射光线和法线在同一入射面内

反射光线的波长、频率、波速都跟入射光保持一致

入射角等于反射角,即\(\theta_i = \theta_r\)

折射

由Huygens原理,利用Huygens作图法我们可以得到 折射定律 ：

入射光线和折射光线与法线共面

射光线和折射光线分别位于法线的两侧(正折射)

入射角和折射角的正弦之比等于光在两种介质中的速度之比:即斯涅尔定律(Snell's Law) \(n_i\sin\theta_i = n_t\sin\theta_t\)

实际上,产生折射的物理机制是由于两种介质中光的传播方向不一样.

费马原理

不论反射定律、折射定律等,还有光的普遍传播方式,可以从费马原理的角度来看待.

费马原理(Fermat Principle)：一束光在两点间实际所走的路径,是以最短的时间通过的那一条路径.

不妨假设光通过\(m\)层介质,每一层介质的折射率是\(n_m\),光速是\(v_m\),则光传播的总时间为： \[ t = \sum_{i=1}^{m}\cfrac{s_i}{v_i} = \cfrac{1}{c}\sum_{i=1}^{m}n_is_i \] 定义光程为路程与折射率的乘积,总光程为： \[ OPL=\int_{S}^{P}n(s)\mathrm{d}s \] 此时费马原理又可表述为：一束光在两点间实际所走的路径是光程最短的路径.

其更准确的表述为：一束光在两点间实际所走的路径是光程取极值的所有可能路径.

费马原理只谈到路径,而不管所沿的方向,光线从\(P\)点到\(S\)点所走的与从\(S\)点到\(P\)点所走的是同一条路线,此即可逆性原理.

菲涅尔公式

Huygens原理与Feymat原理可以预测折射光与反射光的传播方向,但是没办法求出反射光,折射光的强度.基于Maxwell方程组的计算可以精确计算反射光,折射光的强度.

考虑横电偏振模式(TE),此时磁场平行于入射面,电场垂直入射面.界面没有自由电荷与传导电流,边界条件可以写为： \[ \begin{cases} E_i + E_r = E_t \\ B_i\cos\theta_i + B_r\cos\theta_r = B_t\cos\theta_t \end{cases} \] 又有: \[ B_i = \cfrac{E_i}{v_i},B_r = \cfrac{E_r}{v_r},B_t = \cfrac{E_t}{v_t} \] 方程组可以改写为: \[ \begin{cases} E_i + E_r = E_t \\ n_iE_i\cos\theta_i +n_rE_r\cos\theta_r = n_tE_t\cos\theta_t \end{cases} \] 定义反射系数\(r\),透射系数\(t\)如下： \[ r = \cfrac{E_r}{E_i},t=\cfrac{E_t}{E_i} \] 联立以上方程组,有： \[ r = \cfrac{n_i\cos\theta_i-n_t\cos\theta_t}{n_i\cos\theta_i+n_t\cos\theta_t},t = 1+r=\cfrac{2n_i\cos\theta_i}{n_i\cos\theta_i+n_t\cos\theta_t} \] TE偏振的反射系数和透射系数分别可以表示为： \[ r_{s} =\cfrac{n_i\cos\theta_i-n_t\cos\theta_t}{n_i\cos\theta_i+n_t\cos\theta_t} = -\cfrac{\sin(\theta_i-\theta_t)}{\sin(\theta_i+\theta_t)}\\ t_{s} =\cfrac{2n_i\cos\theta_i}{n_i\cos\theta_i+n_t\cos\theta_t}=\cfrac{2\sin\theta_t\cos\theta_i}{\sin(\theta_i+\theta_t)} \] 再考虑横磁(TM)偏振模式,此时电场平行于入射面,磁场垂直入射面.界面没有自由电荷与传导电流,边界条件可以写为： \[ \begin{cases} B_i +B_r = B_t \\ E_i\cos\theta_i + E_r\cos\theta_r = E_t\cos\theta_t \end{cases} \] 方程组可以改写为： \[ \begin{cases} n_iE_i +n_rE_r = n_tE_t \\ E_i\cos\theta_i +E_r\cos\theta_r = E_t\cos\theta_t \end{cases} \] 解方程组,TM偏振的反射系数和透射系数分别可以表示为： \[ r_p=\cfrac{n_t\cos\theta_i-n_i\cos\theta_t}{n_t\cos\theta_i+n_i\cos\theta_t}=\cfrac{\tan(\theta_i-\theta_t)}{\tan(\theta_i+\theta_t)} \\ t_p = \cfrac{2n_i\cos\theta_i}{n_t\cos\theta_i+n_i\cos\theta_t} = \cfrac{2\sin\theta_t\cos\theta_i}{\sin(\theta_i+\theta_t)\cos(\theta_i-\theta_t)} \] 讨论几种特殊的入射情况：

正入射(\(\theta_i=0\))时,有: \[ r_s = -r_p = \cfrac{n_i-n_t}{n_i+n_t} \\ t_s = t_p = \cfrac{2n_i}{n_i+n_t} \\ r_p + t_p= 1 \] 掠入射(\(\theta_i=\frac{\pi}{2}\))时,有: \[ r_s = r_p = -1 \\ t_s = t_p = 0 \\ \]

布儒斯特角

在得到Fresnel公式后,考虑反射光的平行分量,有： \[ r_p=\cfrac{\tan(\theta_i-\theta_t)}{\tan(\theta_i+\theta_t)} \] 若入射角与折射角满足下俩关系： \[ \theta_i+\theta_t=\cfrac{\pi}{2} \] 此时有： \[ r_p=0 \] 此时的入射角即为布儒斯特角(Brewster Angle)

全反射

对于内反射\((n_i>n_t)\),当\(\theta_i\)不断增大时,\(\theta_t\)也在不断增大,考虑\(\mathrm{Snell}\) 定律: \[ n_i\sin\theta_i=n_t\sin\theta_t \] 可知\(\theta_t>\theta_i\),故\(\theta_t\)首先达到\(\frac{\pi}{2}\),定义此时的入射角为全反角： \[ \theta_c=\theta_i = \arcsin(\cfrac{n_t}{n_i}) \] 由三角函数的基本关系可知,对于内反射,全反射角总是大于Brewster角.

半波损失

考虑外反射(\(n_i<n_t\))情形

正入射时,我们已有: \[ r_s = -r_p = \cfrac{n_i-n_t}{n_i+n_t} \\ t_s = t_p = \cfrac{2n_i}{n_i+n_t} \\ r_p + t_p= 1 \] 我们可推断出: \[ r_p > 0,r_s <0 \] 说明不论反射,折射,电场均会反向

掠入射时,我们有 \[ r_s = r_p = -1 \\ t_s = t_p = 0 \\ \] 同样可说明电场均会反向

由此我们可推断出：对于外反射,不论正入射掠入射,反射电场均会反向,即存在\(\pi\)的相位,此即半波损失.

同理,对于内反射我们同样可以证明,正入射不存在半波损失.

反射率与透射率

在Chapter 1 我们引入了光强的概念： \[ I = \langle S\rangle_T = \cfrac{1}{2}nc\varepsilon_0E_0^2 \] 定义反射率为反射光与入射光功率的比值： \[ R = \cfrac{I_rA\cos\theta_r}{I_iA\cos\theta_i} = \cfrac{I_r}{I_i} = r^2 \] 定义透射率为折射光与入射光功率的比值 \[ T = \cfrac{I_tA\cos\theta_t}{I_iA\cos\theta_i} = (\cfrac{n_t\cos\theta_t}{n_i\cos\theta_i})t^2 \] 对于无损耗介质,由能量守恒关系,显然有: \[ R+T=1 \]

全内反射

对于内反射,当\(θ_i≥θ_c\)时,发生全反射.在内反射时发生的全反射,称为全内反射.

发生全反射时,仅考虑反射光与入射光,则边界条件不会满足,此时折射角失去原有的意义.

从\(\mathrm{Snell}\)定律出发： \[ n_i\sin\theta_i = n_t\sin\theta_t \Rightarrow \cos\theta_t = \sqrt{1-\cfrac{n_i^2\sin^2\theta_i}{n_t^2}} \\ \theta_i>\theta_t \Rightarrow\cos\theta_t = i\sqrt{\cfrac{n_i^2\sin^2\theta_i}{n_t^2}-1}=i\alpha \] 透射波的电场可以写为: \[ \vec{\mathbf{E_t}} = \vec{\mathbf{E_{0t}}}\exp[i(\vec{k_t}\cdot\vec{r}-\omega t)] \\ \vec{k_t}\cdot\vec{r} = k_t\sin\theta_tx+k_t\cos\theta_ty \] 带入可得电场最终的形式: \[ \vec{\mathbf{E_t}} = \vec{\mathbf{E_{0t}}}\exp(-\beta y)\exp[i(n_ik_0\sin\theta_ix-\omega t)] \\ \beta = k_t\alpha \] 折射光沿\(y\)方向指数衰减,这种光波成为隐逝波(Evanescent Wave),光场分布为隐逝场.

金属的光学性质

复介电常数

金属中存在自由电子,存在传导电流,此时Maxwell方程组为: \[ \nabla \times \vec{\mathbf{H}} = \varepsilon_0 \varepsilon_r\cfrac{\partial \vec{\mathbf{E}}}{\partial t}+\sigma \vec{\mathbf{E}} \] 考虑光波是简谐波,则有： \[ \nabla \times \vec{\mathbf{H}} = -i\omega\varepsilon_0 \varepsilon_r \vec{\mathbf{E}}+\sigma \vec{\mathbf{E}} \] 定义复介电常数\(\tilde{\varepsilon_r}\),上式变为： \[ \nabla \times \vec{\mathbf{H}} = -i\omega\varepsilon_0 \tilde{\varepsilon_r} \vec{\mathbf{E}} \] 复介电常数可以写为： \[ \tilde{\varepsilon_r} = \varepsilon_r + i\cfrac{\sigma}{\omega\varepsilon_0} \]

趋肤深度

定义负折射率为： \[ \tilde{n}=n^{'}+in^{''} \] 根据Maxwell关系： \[ \tilde{n}^2 = \tilde{\varepsilon_r} = \varepsilon_r + i\cfrac{\sigma}{\omega\varepsilon_0} \] 解得： \[ n^{'} = [\cfrac{\varepsilon_r}{2}\sqrt{1+(\cfrac{\sigma}{\omega\varepsilon_0\varepsilon_r})^2}+1]^{1/2},n^{''} = [\cfrac{\varepsilon_r}{2}\sqrt{1+(\cfrac{\sigma}{\omega\varepsilon_0\varepsilon_r})^2}-1]^{1/2} \] 在良导体近似条件\(\frac{\sigma}{\omega\varepsilon_0\varepsilon_r} >>1\)的情况下,我们有 \[ n^{'}=n^{''} = (\cfrac{\sigma}{2\omega\varepsilon_0})^{1/2} \] 金属中的波矢可以表示为： \[ k=\tilde{n}k_0=(\cfrac{\omega\mu_0\sigma}{2})^{1/2}+i(\cfrac{\omega\mu_0\sigma}{2})^{1/2}=k^{'}+ik^{''} \] 此时金属中的电场为： \[ E = E_0\exp(ik^{'}z)\exp(-k^{''}z) \] 光强为： \[ I = I_0\exp(-2k^{''}z) \] 定义金属的趋肤深度为： \[ \delta = \cfrac{1}{2k^{''}} = (2\omega\mu_0\sigma)^{-1/2} \]

金属的色散

考虑Chapter 1提到的电偶极子模型,金属中无势能项故\(\omega_0=0\),此时微分方程变为： \[ m_e \frac{d^2 x}{d t^2}+\gamma m_e \frac{d x}{d t} =-eE_0\exp(-i\omega t) \] 令\(x = x_0\exp(-i\omega t)\),解得： \[ x(t) = \frac{eE(t)}{m}\cfrac{1}{\omega^2+i\omega\gamma},P=-Nex=-N\frac{e^2E(t)}{m}\cfrac{1}{\omega^2+i\omega\gamma} \] 此时复介电常数为： \[ \varepsilon_r = 1+\cfrac{P}{\varepsilon_0E} = 1-\cfrac{\omega_p^2}{\omega(\omega+i\gamma)} \] 其中\(\omega_p\)是等离子体频率(plasma frequency).

金属表面的反射

考虑一束光正入射到金属表面,其反射率为: \[ R = |\cfrac{n_i-n_t}{n_i+n_t}|^2,n_i=1,n_t=\tilde{n} \] 此时有： \[ R = \cfrac{(n^{'}-1)^2+n^{''2}}{(n^{'}+1)^2+n^{''2}} \]
