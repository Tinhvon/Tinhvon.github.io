Chapter2-03求导方法

导数与微分

求导方法

下面我们介绍较复杂函数的求导方法.

隐函数求导法

首先给出隐函数的定义:

Definition:设关于\(x,y\)的代数式\(F(x,y)\),且方程\(F(x,y )=0\)可以确定\(y\)是\(x\)的函数,则称这种方式表示的函数是隐函数.

Note1:相比于隐函数,显函数一般可以用\(y=f(x)\)直接表示.

对于隐函数\(F(x,y)=0\),求导数\(y'\)的方法如下:

  • 关于\(x\)求导,其中\(y\)是\(x\)的函数.
  • 解出\(y'\)即可.

下面给一个例子来应用这种方法:

Example 1:已知\(y = \cos{(x+y)}\),\(y\)是\(x\)的函数,求\(y'\).

Solution 对等式两边求导有:\(y' = -\sin(x+y)(1+y')\)

解得\(y'=-\cfrac{\sin(x+y)}{1+\sin(x+y)}\)

Example 2:已知\(e^y-e^{-x}+xy=0\),求\(y'\).

Solution 对等式两边求导有:\(y'e^{y}+e^{-x}+y+xy'=0\)

解得\(y'=-\cfrac{y+e^{-x}}{x+e^{y}}\)

对数求导法

当所求函数可以写成函数相乘(特别是指数相乘)的形式时,常用对数求导法,方法如下:

设函数\(y=f_1(x)\cdot f_2(x)\cdot ...\cdot f_n(x)\)

对两边取对数\(\ln{y} = \ln f_1(x)+\ln f_2(x)+...+\ln f_n(x)\)

两边求导,即有:\(\cfrac{y'}{y} = \cfrac{f_1'(x)}{f_1(x)}+\cfrac{f_2'(x)}{f_2(x)}+...+\cfrac{f_n'(x)}{f_n(x)}\)

可得\(y = \sum\limits_{k=1}^{n}f_1(x)\cdot ...\cdot f_k'(x)\cdot ... f_n(x)\)

当所求函数形式为指数形式时,对数求导法可以发挥其最大的效用.给出一个应用的例子:

Example 1:已知函数\(y=(\sin{x})^{\cos{x}}+(\cos{x})^{\sin{x}}\),求\(y'\).

Solution 两边取对数:\(\ln{y} = \cos{x}\ln{\sin{x}} + \sin{x}\ln{\cos{x}}\)

两边求导:\(\cfrac{y'}{y} = \cos{x}\ln{\sin{x}}-\sin{x}\ln{\sin{x}}+\cfrac{\cos^2x}{\sin{x}}-\cfrac{\sin^2{x}}{\cos{x}}\)

最终化简即可

参数方程求导法

当函数是用参数方程表示的形式时,如: \[ \begin{cases} x = x(t)\\ y = y(t) \end{cases},a\leq t \leq b \] 其求导方法本质不变.对于参数方程表示的函数,求其导数,有以下参数方程求导法:

\(\cfrac{\mathrm{d}y}{\mathrm{d}x} = \cfrac{\mathrm{d}y}{\mathrm{d}t}\cdot \cfrac{\mathrm{d}t}{\mathrm{d}x} = \cfrac{\mathrm{d}y/\mathrm{d}t}{\mathrm{d}x/\mathrm{d}t}\)

给出几个例子:

Example 1:已知摆线的参数方程为:\(\begin{cases}x = a(t-\sin{t})\\y=a(1-\cos{t})\end{cases},0\leq t\leq 2\pi\),求其导数.

Solution 易知:\(\cfrac{\mathrm{d}y}{\mathrm{d}x} = \cfrac{y'(t)}{x'(t)} = \cfrac{\sin{t}}{1-\cos{t}}\)

Example 2:极坐标系下,心型线的极坐标方程为\(r=a(1+\cos{\theta})\),求\(\cfrac{\mathrm{d}y}{\mathrm{d}x}\).

Solution 直角坐标与极坐标之间的代换关系为:\(\begin{cases}x=r\cos{\theta}\\y=r\sin{\theta}\end{cases}\)

化简有:\(\begin{cases}x=a(1+\cos{\theta})\cos{\theta}\\y=a(1+\cos{\theta})\sin{\theta}\end{cases}\)

由参数方程求导法:\(\cfrac{\mathrm{d}y}{\mathrm{d}x} = \cfrac{y'(\theta)}{x'(\theta)} = -\cfrac{\cos2\theta+\cos{\theta}}{\sin{2\theta+\sin{\theta}}}\).
