Chapter2-03求导方法

导数与微分

求导方法

下面我们介绍较复杂函数的求导方法.

隐函数求导法

首先给出隐函数的定义:

Definition:设关于x,yx,yx,y的代数式F(x,y)F(x,y)F(x,y),且方程F(x,y)=0F(x,y )=0F(x,y)=0可以确定yyy是xxx的函数,则称这种方式表示的函数是隐函数.

Note1:相比于隐函数,显函数一般可以用y=f(x)y=f(x)y=f(x)直接表示.

对于隐函数F(x,y)=0F(x,y)=0F(x,y)=0,求导数y′y'y′的方法如下:

  • 关于xxx求导,其中yyy是xxx的函数.
  • 解出y′y'y′即可.

下面给一个例子来应用这种方法:

Example 1:已知y=cos⁡(x+y)y = \cos{(x+y)}y=cos(x+y),yyy是xxx的函数,求y′y'y′.

Solution 对等式两边求导有:y′=−sin⁡(x+y)(1+y′)y' = -\sin(x+y)(1+y')y′=−sin(x+y)(1+y′)

解得y′=−sin⁡(x+y)1+sin⁡(x+y)y'=-\cfrac{\sin(x+y)}{1+\sin(x+y)}y′=−1+sin(x+y)sin(x+y)​

Example 2:已知ey−e−x+xy=0e^y-e^{-x}+xy=0ey−e−x+xy=0,求y′y'y′.

Solution 对等式两边求导有:y′ey+e−x+y+xy′=0y'e^{y}+e^{-x}+y+xy'=0y′ey+e−x+y+xy′=0

解得y′=−y+e−xx+eyy'=-\cfrac{y+e^{-x}}{x+e^{y}}y′=−x+eyy+e−x​

对数求导法

当所求函数可以写成函数相乘(特别是指数相乘)的形式时,常用对数求导法,方法如下:

设函数y=f1(x)⋅f2(x)⋅...⋅fn(x)y=f_1(x)\cdot f_2(x)\cdot ...\cdot f_n(x)y=f1​(x)⋅f2​(x)⋅...⋅fn​(x)

对两边取对数ln⁡y=ln⁡f1(x)+ln⁡f2(x)+...+ln⁡fn(x)\ln{y} = \ln f_1(x)+\ln f_2(x)+...+\ln f_n(x)lny=lnf1​(x)+lnf2​(x)+...+lnfn​(x)

两边求导,即有:y′y=f1′(x)f1(x)+f2′(x)f2(x)+...+fn′(x)fn(x)\cfrac{y'}{y} = \cfrac{f_1'(x)}{f_1(x)}+\cfrac{f_2'(x)}{f_2(x)}+...+\cfrac{f_n'(x)}{f_n(x)}yy′​=f1​(x)f1′​(x)​+f2​(x)f2′​(x)​+...+fn​(x)fn′​(x)​

可得y=∑k=1nf1(x)⋅...⋅fk′(x)⋅...fn(x)y = \sum\limits_{k=1}^{n}f_1(x)\cdot ...\cdot f_k'(x)\cdot ... f_n(x)y=k=1∑n​f1​(x)⋅...⋅fk′​(x)⋅...fn​(x)

当所求函数形式为指数形式时,对数求导法可以发挥其最大的效用.给出一个应用的例子:

Example 1:已知函数y=(sin⁡x)cos⁡x+(cos⁡x)sin⁡xy=(\sin{x})^{\cos{x}}+(\cos{x})^{\sin{x}}y=(sinx)cosx+(cosx)sinx,求y′y'y′.

Solution 两边取对数:ln⁡y=cos⁡xln⁡sin⁡x+sin⁡xln⁡cos⁡x\ln{y} = \cos{x}\ln{\sin{x}} + \sin{x}\ln{\cos{x}}lny=cosxlnsinx+sinxlncosx

两边求导:y′y=cos⁡xln⁡sin⁡x−sin⁡xln⁡sin⁡x+cos⁡2xsin⁡x−sin⁡2xcos⁡x\cfrac{y'}{y} = \cos{x}\ln{\sin{x}}-\sin{x}\ln{\sin{x}}+\cfrac{\cos^2x}{\sin{x}}-\cfrac{\sin^2{x}}{\cos{x}}yy′​=cosxlnsinx−sinxlnsinx+sinxcos2x​−cosxsin2x​

最终化简即可

参数方程求导法

当函数是用参数方程表示的形式时,如:

{x=x(t)y=y(t),a≤t≤b\begin{cases} x = x(t)\\ y = y(t) \end{cases},a\leq t \leq b {x=x(t)y=y(t)​,a≤t≤b

其求导方法本质不变.对于参数方程表示的函数,求其导数,有以下参数方程求导法:

dydx=dydt⋅dtdx=dy/dtdx/dt\cfrac{\mathrm{d}y}{\mathrm{d}x} = \cfrac{\mathrm{d}y}{\mathrm{d}t}\cdot \cfrac{\mathrm{d}t}{\mathrm{d}x} = \cfrac{\mathrm{d}y/\mathrm{d}t}{\mathrm{d}x/\mathrm{d}t}dxdy​=dtdy​⋅dxdt​=dx/dtdy/dt​

给出几个例子:

Example 1:已知摆线的参数方程为:{x=a(t−sin⁡t)y=a(1−cos⁡t),0≤t≤2π\begin{cases}x = a(t-\sin{t})\\y=a(1-\cos{t})\end{cases},0\leq t\leq 2\pi{x=a(t−sint)y=a(1−cost)​,0≤t≤2π,求其导数.

Solution 易知:dydx=y′(t)x′(t)=sin⁡t1−cos⁡t\cfrac{\mathrm{d}y}{\mathrm{d}x} = \cfrac{y'(t)}{x'(t)} = \cfrac{\sin{t}}{1-\cos{t}}dxdy​=x′(t)y′(t)​=1−costsint​

Example 2:极坐标系下,心型线的极坐标方程为r=a(1+cos⁡θ)r=a(1+\cos{\theta})r=a(1+cosθ),求dydx\cfrac{\mathrm{d}y}{\mathrm{d}x}dxdy​.

Solution 直角坐标与极坐标之间的代换关系为:{x=rcos⁡θy=rsin⁡θ\begin{cases}x=r\cos{\theta}\\y=r\sin{\theta}\end{cases}{x=rcosθy=rsinθ​

化简有:{x=a(1+cos⁡θ)cos⁡θy=a(1+cos⁡θ)sin⁡θ\begin{cases}x=a(1+\cos{\theta})\cos{\theta}\\y=a(1+\cos{\theta})\sin{\theta}\end{cases}{x=a(1+cosθ)cosθy=a(1+cosθ)sinθ​

由参数方程求导法:dydx=y′(θ)x′(θ)=−cos⁡2θ+cos⁡θsin⁡2θ+sin⁡θ\cfrac{\mathrm{d}y}{\mathrm{d}x} = \cfrac{y'(\theta)}{x'(\theta)} = -\cfrac{\cos2\theta+\cos{\theta}}{\sin{2\theta+\sin{\theta}}}dxdy​=x′(θ)y′(θ)​=−sin2θ+sinθcos2θ+cosθ​.
