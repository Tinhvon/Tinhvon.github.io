微积分-Chapter2-导数与微分-微分

导数与微分

微分

在系统地谈论导数后，我们最后来谈论一下微分。在谈论微分之前，我们先讨论一下局部线性化。

局部线性化

根据下图，设直线\(T\)与\(y=f(x)\)在\(x=x_0\)处相切，则\(T:f(x)-f(x_0) = f'(x_0)(x-x_0)\).

如图，用\(KN\)替代\(PN\),则有\(PN = \Delta y = f(x_0+\Delta x)-f(x_0),KN = f'(x_0)(x-x_0)\),联立后有\(f(x_0+\Delta x)-f(x_0)=f'(x_0)\Delta x+o(x-x_0)\),其中\(PK = o(x-x_0) = o(\Delta x)\),此即化曲为直，即局部线性化。

基本概念

然后我们严格地给出微分的定义：

Definition:设\(y=f(x)\)在\((a,b)\)有定义，且\(\Delta y = f(x+\Delta x)-f(x) = Ax+o(\Delta x),\Delta x \to 0\).其中\(A\)与\(\Delta x\)无关，则称\(y=f(x)\)在\(x\)可微，并称\(A\Delta x\)为\(y=f(x)\)在\(x\)的微分，记作\(\mathrm{d}y = A\Delta x\).

Note 1:其中我们称\(A\Delta x\)是\(\Delta y\)的线性主部，故我们有\(\Delta y-\mathrm{d}y = o(\Delta x),\Delta x \to 0\).

Note 2:对一元函数来说，\(y=f(x)\)在\(x=x_0\)处可微\(\Leftrightarrow y = f(x)\)在\(x = x_0\)可导，这里直接套用定义证明即可，具体证明略。

Note 3:对\(y=f(x),\mathrm{d}y = f'(x)\Delta x = f'(x)\mathrm{d}x\).

Note 4:微分公式及其运算法则，与导数的运算法则相同。

Example 1:已知\(y=\sin{x^2}\),求\(\mathrm{d}y\).

Solution \(\mathrm{d}y = 2x\cos{x^2}\mathrm{d}x\)

Note 5:微分的几何意义：\(y=f(x)\)在\(P(x,y)\)处的微分，即曲线\(y=f(x)\)在\(P\)切线在\(P\)点纵坐标的改变量。

Note 6:一阶微分具有形式不变性，即\(y=f(u),u=g(x)\Rightarrow y=f[g(x)]\),则有\(\mathrm{d}y = f'[g(x)]g'(x)\mathrm{d}x = f'(u)\mathrm{d}u\) .但是高阶微分不具有微分形式不变性。

近似计算

利用微分的概念，我们可以近似计算一些复杂的代数式。原理如下：

Theorem:根据局部线性化原理，我们有\(f(x) = f(x_0)+f'(x_0)(x-x_0)\).

这里我们给出一个例子：

Example :近似计算\(\sqrt[3]{29}\)

Solution 设\(f(x) = \sqrt[3]{x}\),则\(f(29) = f(27)+f'(27)\times(29-27) = 3+\cfrac{2}{27} = 3.0741\)
