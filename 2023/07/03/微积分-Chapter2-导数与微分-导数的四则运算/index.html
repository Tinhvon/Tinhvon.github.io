微积分-Chapter2-导数与微分-导数的四则运算

导数与微分

导数的运算

上一篇博文中,我们介绍了导数的基本概念,这篇博文我们介绍一下导数的运算.

四则运算

我们不加证明地给出导数的四则运算法则

Theorem:设\(f(x),g(x)\)是可导函数,则其满足以下运算规则:

  • \([\alpha f(x)+\beta g(x)]' = \alpha f'(x)+\beta g'(x)\)
  • \([f(x)g(x)]' = f'(x)g(x)+f(x)g'(x)\)
  • \([\cfrac{f(x)}{g(x)}]' = \cfrac{f'(x)g(x)-f(x)g'(x)}{g^2(x)},(g(x)\ne 0)\)

根据导数运算的除法规则,我们可以得到一个特殊的导数公式: \[ (\tan x)' = [\cfrac{\sin x}{\cos x}]' = \cfrac{\cos^2 x+\sin^2 x }{\cos ^2x} = \sec^2x \] 关于导数四则运算法则的证明,直接套用导数的定义即可,这里不再讨论.

反函数求导

对于有反函数的函数求导,有反函数求导定律:

Theorem:设\(y=f(x)\)是严格单调连续函数,在\(x\)点可导且\(f'(x) \ne 0\),则其反函数\(x = g(y)\)在\(x\)的对应点\(y\)可导.(原函数的导数等于反函数导数的倒数).数学表示形式为:\(g'(y) = \cfrac{1}{f'(x)}\)

根据反函数求导法则,我们可以求反三角函数的导数,以\(y = \arcsin x\)为例: \[ x = \sin y,\cfrac{\mathrm{d}x}{\mathrm{d}y} = \cos y = \sqrt{1-x^2}\\ \cfrac{\mathrm{d}y}{\mathrm{d}x} = \cfrac{1}{\cfrac{\mathrm{d}x}{\mathrm{d}y}} = \cfrac{1}{\sqrt{1-x^2}} \] 同理我们便可以得到反三角函数的导数公式: \[ \begin{aligned} (\arcsin{x})' &= \cfrac{1}{\sqrt{1-x^2}} \\ (\arccos{x})' &= -\cfrac{1}{\sqrt{1-x^2}} \\ (\arctan{x})' &= \cfrac{1}{1+x^2}\\ (\mathrm{arccot}{x})' &= -\cfrac{1}{1+x^2} \end{aligned} \]

复合函数求导

我们在这里不加证明地引出复合函数求导的链式法则“:

Theorem:设\(f(x),g(x)\)可导,则\(f[g(x)]\)可导,且\(\{f[g(x)]\}' = \cfrac{\mathrm{d}f(u)}{\mathrm{d}u}\cdot \cfrac{\mathrm{d}u}{\mathrm{d}x},u=g(x)\).

根据链式法则,我们可以求一些复合函数的导数了.给出几个例子:

Example 1:\(f(x)=e^{\sin ^2 x}\),求其导数:

Solution \(f'(x) = 2\sin^2x\cos xe^{\sin ^2x}\)
