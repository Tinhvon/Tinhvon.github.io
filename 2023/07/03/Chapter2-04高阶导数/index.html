导数与微分

高阶导数

此前我们已经系统地讨论了导数的概念,运算法则,求导方法.但这些都仅仅讨论了一阶导数(即求导次数为1).这篇文章里,我们将接触高阶导数.

常见函数的高阶导数

给出几个高阶导数的例子：

  • $(e^x)^{(n)} = e^x,(e^{ax+b})^{(n)} = a^ne^{ax+b}$
  • $(x^{\alpha})^n = \alpha(\alpha-1)…(\alpha-n+1)x^{\alpha-n}$
  • $(\ln(1+x))^{(n)} = (-1)^{n-1}\cfrac{(n-1)!}{(1+x)^n},\ln(ax+b))^{(n)} = (-1)^{n-1}\cfrac{(n-1)!a^n}{(ax+b)^n}$
  • $(\sin{x})^{(n)} = \sin({\cfrac{n\pi}{2}+x)},[\sin{(ax+b})]^{(n)} = a^n\sin({\cfrac{n\pi}{2}+ax+b})$
  • $(\cos x)^{(n)} = \cos({\cfrac{n\pi}{2}+x)},[\cos{(ax+b})]^{(n)} = a^n\cos({\cfrac{n\pi}{2}+ax+b})$
  • $(\cfrac{1}{1-x})^{(n)} = \cfrac{n!}{(1-x)^{n+1}},(\cfrac{1}{1+x})^{(n)} = \cfrac{(-1)^nn!}{(1-x)^{n+1}},(\cfrac{1}{ax+b})^{(n)} = \cfrac{(-1)^n a^n n!}{(ax+b)^{n+1}}$

以上给出的是一些基本函数的$n$阶导数,其可用来求基本初等函数的高阶导数,例子如下：

Example 1:已知函数$y = \cfrac{x^3}{1-x}$,求$y^{(n)}$

Solution $y = \cfrac{1}{1-x}-(x^2+x+1)$

分情况讨论：

  • $y’ = \cfrac{1}{(1-x)^2}-2x-1$
  • $y’’ = \cfrac{2}{(1-x)^3}-2$
  • $y^{(n)} = \cfrac{n!}{(1-x)^{n+1}}(n \geq 3)$

Example 2: 已知函数$y=\cfrac{1}{x^2+2x-3}$,求$y^{(n)}$

Solution $y = \cfrac{1}{4}(\cfrac{1}{x-1}-\cfrac{1}{x+3})$

$y^{(n)} = \cfrac{1}{4}[\cfrac{n!}{(1-x)^{n+1}} - \cfrac{(-1)^nn!}{(3+x)^{n+1}}]$

隐函数的高阶导数

此前我们讨论过一阶导数情况下的隐函数求导法,若想求得其高阶导数,则两边求$n$阶导即可.

Example 1:已知$x-y+\cfrac{1}{2}\sin{y} = 0$,求$y’’$

Solution 先对等式两边求导,有：$1-y’+\cfrac{1}{2}y’\cos{y} = 0$

解得:$\cfrac{\mathrm{d}y}{\mathrm{d}x} = \cfrac{2}{2-\cos{y}}$

再求一次导,有:$-y’’+\cfrac{1}{2}y’’\cos{y}-\cfrac{1}{2}\sin{y}(y’)^2=0$

解得:$\cfrac{\mathrm{d}^2y}{\mathrm{d}x^2} = \cfrac{-4\sin{y}}{(2-\cos{y})^3}$

莱布尼兹公式

我们不加证明地给出高阶导数的莱布尼兹公式：

Theorem:设$u(x),v(x)$有$n$阶导数,则：$[u(x)v(x)]^{(n)} = \sum\limits_{k=0}^nC_{n}^{k}u^{(n-k)}(x)v^{(k)}(x)$,此即高阶导数的莱布尼兹公式.

下面给出一个应用的例子：

Example 1:已知函数$f(x) = x^2e^x$,求$f^{(n)}(x)$

Solution 根据莱布尼兹公式,有$f^{(n)}(x) = x^2e^x+2nxe^x+n(n-1)e^x$.

导数极限定理

我们不加证明地给出导数极限定理：

Theorem:设$f(x)$在$N_0(x_0,\delta)$可导,在$x_0$连续,且$\lim\limits_{x\to x_0}f’(x) = A$,则$f’(x_0)=A$.

直接给出例题：

Example 1:已知$f(x) = \begin{cases}e^x,x<0 \\ ax^2+bx+c,x \geq 0\end{cases}$,现已知$f’’(0)$存在,求$a,b,c$.

Solution $f’(x) = \begin{cases}e^x,x<0 \\ 2ax+b,x \geq 0\end{cases}$,根据连续性可知$b=1$.

$f’’(x) = \begin{cases}e^x,x<0 \\ 2a,x \geq 0\end{cases}$,根据连续性可知$a = \cfrac{1}{2},c=1$

综上:$a=\cfrac{1}{2},b=1,c=1$
