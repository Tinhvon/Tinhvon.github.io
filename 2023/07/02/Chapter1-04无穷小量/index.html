Chapter1-04无穷小量

极限与连续

无穷小量

无穷小量的概念

首先给出无穷小量的定义:

Definition:若$x\to x_0$时,$\alpha(x) \to 0$,则称 $ \alpha (x)$为 $x\to x_0$的无穷小量.

数学语言表述为:$\lim\limits_{x \to x_0}f(x) = A \Leftrightarrow f(x) = A +\alpha ,\lim\limits_{x \to x_0} \alpha =0$

无穷小量的推论

给出无穷小的定义后,我们对无穷小做进一步讨论

Note1:

  • 有限个无穷小之和为无穷小
  • 有限个无穷小之积为无穷小
  • 有界量与无穷小之积也为无穷小
  • 无穷个无穷小之和不一定为无穷小,如$x_n = \cfrac{1}{n},n\to \infty$
  • 无穷个无穷小之积不一定为无穷小

以上证明略,由Note1的第三点可以得知$\lim\limits_{x \to 0}x\sin(\cfrac{1}{x}) = 0$

讨论无穷小和与积的性质后,我们暂未搞懂应当怎样描述无穷小的大小,先给出推论:

Note2:

  • $\lim\limits_{x \to x_0}\cfrac{\alpha(x)}{\beta(x)}= 0$,称$\alpha(x)$是$\beta(x)$的高阶无穷小,记作$\alpha(x) = o(\beta(x))$
  • $\lim\limits_{x \to x_0}\cfrac{\alpha(x)}{\beta(x)}= C \ne 0$,称$\alpha(x)$是$\beta(x)$的同阶无穷小,记作$\alpha(x) \sim C\beta(x)$,若$C = 1$,称$\alpha(x)$是$\beta(x)$的等价无穷小,记作$\alpha(x) \sim \beta(x),x\to x_0$

在掌握无穷小比较的语言后,我们可以建立一个“坐标系”,将一个简单的无穷小作为参照物,进而比较其他复杂无穷小量的阶数,因此我们引入基本无穷小的定义:

Definition:当$x\to x_0$时,称$(x-x_0)$为基本无穷小,若$\alpha (x) \sim A(x - x_0)^{r}(A \ne 0,r > 0)$则称$\alpha(x)$为$r$阶无穷小,并称$ A(x - x_0)^{r}$为$\alpha(x)$的主部,且$\alpha (x) = A(x-x_0)^{r}+o((x-x_0)^r)$

给出几个例子:

Example 1:$\alpha(x) = \cfrac{2x}{1+x^2}(x \to 0),\alpha(x) \sim 2x$

Example 2:$\beta(x) = \ln x^2(x\to1),\beta(x) =\ln x^2 = \ln(1+x^2-1)\sim (x^2-1) = (x+1)(x-1) = 2(x-1)$

无穷小代换

引入无穷小这一概念后,我们求极限的方法又多了一种,记无穷小代换,其数学语言叙述如下:

Theorem:当$x\to x_0$时,$\alpha \sim \alpha’,\beta \sim \beta’$,则有$\lim\limits_{x\to x_0}\cfrac{\alpha}{\beta} =\lim\limits_{x\to x_0}\cfrac{\alpha’}{\beta’} $

给出几个例子:

  • 与$x$等价的无穷小(默认$x \to x_0$)

    $x,\sin x,\arcsin x,\tan x,\arctan x,\ln(1+x),e^x-1,n(\sqrt[n]{1+x}-1)$

  • $1-\cos x\sim \cfrac{1}{2}x^2,x-\sin x \sim \cfrac{1}{6}x^3$

Example 2:计算$\lim\limits_{x\to 0}\cfrac{\sqrt{4+x^2}}{\sqrt{9+x^3}}\cdot \cfrac{\arcsin x}{e^x-1}\cdot \cfrac{\tan x}{\sin x}\cdot \cfrac{\ln (1+x)}{\arctan x}$

Solution 根据等价无穷小代换,有:

$\lim\limits_{x\to 0}\cfrac{\sqrt{4+x^2}}{\sqrt{9+x^3}}\cdot \cfrac{\arcsin x}{e^x-1}\cdot \cfrac{\tan x}{\sin x}\cdot \cfrac{\ln (1+x)}{\arctan x} = \lim\limits_{x\to 0}\cfrac{\sqrt{4+x^2}}{\sqrt{9+x^3}}\cdot \cfrac{x}{x}\cdot \cfrac{x}{x}\cdot \cfrac{x}{x} = \cfrac{2}{3}$

无穷大

给出无穷大的定义:

Definition:若$\lim\limits_{x\to x_0}L(x) = \infty$,则称$L(x)$为$x\to x_0$的无穷大(量).

对于无穷大量,我们有一个常用的定理:

Theorem:当$x\to x_0$时,$L(x)$为无穷大量,则$\cfrac{1}{L(x)}$为无穷小量.

最后再给出一个例题:

Example 1:计算$\lim\limits_{x \rightarrow 0} \cfrac{(1+m x)^n-(1+n x)^m}{x^2} $

Solution $\lim\limits_{x \rightarrow 0} \cfrac{(1+m x)^n-(1+n x)^m}{x^2} =\lim\limits_{x \rightarrow 0}\frac{\frac{1}{2} n(n-1) m^2 x^2+o\left(x^2\right)-\frac{1}{2} m(m-1)n^2 x^2-o(x^2)}{x^2}=\lim\limits_{x \rightarrow 0} \frac{\frac{1}{2}\left[\left(n(n-1) m^2-m(m-1) n^2\right] x^2+0\left(x^2\right)\right.}{x^2}=\frac{1}{2} m n(n-m) $

等价无穷小代换及应用

等价无穷小代换是求解极限的一种常用的方法,下面我们给出几道例题来细细体验一下这种方法:

Example 1:计算$\lim\limits_{x\to 0}(\cfrac{x+\sin x}{x})$

Solution $I = \lim\limits_{x\to 0}(\cfrac{x+x}{x}) =2$

Example 2:计算$\lim\limits_{x\to 0}(\cfrac{x-\sin x}{x^3})$

Solution $I = \lim\limits_{x\to 0}(\cfrac{\frac{1}{6}x^3}{x^3}) = \cfrac{1}{6}$

针对前两道例题,我们可以给出一个推论:

Theorem:当$x\to x_0$时,若$\alpha \sim \alpha’,\beta \sim \beta’,r\sim r’$,且$\lim\limits_{x\to x_0}\cfrac{\alpha}{r} \ne -1$,则有$\lim\limits_{x\to x_0}\cfrac{\alpha +r}{\beta} = \lim\limits_{x\to x_0}\cfrac{\alpha’ +r’}{\beta’}$

得到这个推论后,我们再来看下列例题:

Example 3:计算$\lim\limits_{x\to 0}\cfrac{x\cos x-x+x^3+o(x^3)}{x^3}$

Solution 根据以上推论,$I = \lim\limits_{x\to 0}[\cfrac{x\cos x-x}{x^3}+\cfrac{x^3+o(x^3)}{x^3}] = \cfrac{1}{2}$

Example 4:计算$\lim\limits_{x \to x_0}\cfrac{\tan^3(2x)}{x^4}(1-\cfrac{x}{e^x-1})$

Solution $I = 8\lim\limits_{x\to0}[\cfrac{e^x-x-1}{x(e^x-1)}] = 8\lim\limits_{x\to 0}\cfrac{\frac{1}{2}x^2+o(x^2)}{x^2}=4$

Example 5:计算$\lim\limits_{x \rightarrow 0} \cfrac{3 \sin x+x^2 \cos \frac{1}{x}}{(1+\cos x) \ln (1+x)}$

Solution $I = \lim\limits_{x \rightarrow 0}\left(\cfrac{3 \sin x+x \cos x}{2 x}\right)=\cfrac{3}{2}$

Example 6:计算$\lim\limits_{x \rightarrow 0}(\cos x)^{\frac{1}{\ln \left(1+x^2\right)}}$

Solution $I = \lim\limits_{x \rightarrow 0}\left\{(1+\cos x-1)^{\frac{1}{\cos x-1}}\right\}^{\frac{\cos x-1}{\ln \left(1+x^2\right)}}=e^{-\frac{1}{2}}$

Example 7:计算$\lim\limits_{x \rightarrow 0} \cfrac{\sqrt{1+x}+\sqrt{1-x}-2}{x^2}$

Solution

$\begin{align}I &= \lim\limits_{x \rightarrow 0} \cfrac{\sqrt{1+x}-1+\sqrt{1-x}-1}{x^2} \\ & =\lim\limits_{x \rightarrow 0} \cfrac{\frac{x}{\sqrt{1+x}+1}-\frac{x}{\sqrt{1-x}+1}}{x^2} \\ &= \cfrac{1}{4}\lim _{x \rightarrow 0} \cfrac{\sqrt{1-x}-\sqrt{1+x}}{x} \\ &=\cfrac{1}{4} \lim\limits_{x \rightarrow 0} \cfrac{-2 x}{(\sqrt{1-x}+\sqrt{1+x}) x} \\ &= -\cfrac{1}{4}\end{align}$

Example 8:计算$\lim\limits_{x \rightarrow 0} \cfrac{[\sin x-\sin (\sin x)] \sin x}{x^4}$

Solution $I = \lim\limits_{x \rightarrow 0}\cfrac{\sin(x) -\sin (\sin x)}{x^3} = \lim\limits_{x \rightarrow 0}\cfrac{x-\sin x}{x^3}= \cfrac{1}{6}$
