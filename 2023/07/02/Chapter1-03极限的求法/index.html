极限与连续

极限的求法

在讨论完极限的定义和各种性质后,我们主要关心的是如何计算极限,下面介绍一些计算极限的方法:

夹逼定理法

Example1:计算\(\lim\limits_{n \to ∞}\sqrt[n]{1^n+2^n+...+2022^n}\)

Solution 考虑数列\(a_n = \sqrt[n]{1^n+2^n+...+2022^n}\)

则有\(2022^n \leq 1^n+2^n+...+2022^n \leq 2022\times 2022^n\)

即有\(2022 \leq a_n \leq 2022\sqrt[n]{2022}\)

又\(\lim\limits_{n \to ∞}2022 = \lim\limits_{n \to ∞}2022\sqrt[n]{2022} = 2022\)

根据夹逼定理,有\(\lim\limits_{n \to ∞}\sqrt[n]{1^n+2^n+...+2022^n} = 2022\)

Example2:证明:\(\lim\limits_{n \to ∞}\cfrac{n^{k}}{a^{n}}(a > 1) = 0,\lim\limits_{n \to ∞}\cfrac{\ln n}{n}= 0,\lim\limits_{n \to ∞}\cfrac{c^n}{n!}(c > 1) = 0\)（数列增长速度的比较）

Proof

  • 首先证明\(\lim\limits_{n \to ∞}\cfrac{n^{k}}{a^{n}}(a > 1) = 0\)

    记\(a = 1 + \alpha (\alpha > 0) \Rightarrow a^{n} = (1 + \alpha)^{n} > \cfrac{n(n-1)...(n-k)}{(k+1)!}\alpha^{k+1} = An(n-1)...(n-k)\)

    有\(0 < \cfrac{n^{k}}{a^{n}} < \cfrac{n^k}{An(n-1)...(n-k)}\)

    根据夹逼定理:\(\lim\limits_{n \to ∞}\cfrac{n^{k}}{a^{n}}(a > 1) = 0\)

  • 再证明\(\lim\limits_{n \to ∞}\cfrac{\ln n}{n}= 0\)

    记\(\ln n = t,n = e^t\)

    则根据夹逼定理显然有\(\lim\limits_{n \to ∞}\cfrac{\ln n}{n} = \lim\limits_{t \to ∞}\cfrac{t}{e^t} = 0\)

    再证明\(\lim\limits_{n \to ∞}\cfrac{\ln n}{n}= 0\)

  • 最后证明\(\lim\limits_{n \to ∞}\cfrac{c^n}{n!}(c > 1) = 0\)

    记\([c] = k\)

    有\(0 < \cfrac{c^n}{n!} = \cfrac{c\cdot c\cdot ...\cdot c}{1\cdot 2...\cdot k}\cdot \cfrac{c\cdot ... \cdot c\cdot c}{(k+1) ...(n-1)n}\)

    化简有$0 < < $

    根据夹逼定理\(\lim\limits_{n \to ∞}\cfrac{c^n}{n!}(c > 1) = 0\)

Note:当\(n\to ∞\)时,数列增长速度从小到大排列为\(\ln n < n^k < a^n < n! < n^n(a > 1)\)

Example3:证明以下极限\(\lim\limits_{x \to 0^{+}}[x]\cdot \cfrac{1}{x} = 0,\lim\limits_{x \to 0}x \cdot [\cfrac{1}{x}] = 1\)

Proof:

  • 首先证明\(\lim\limits_{x \to 0^{+}}[x]\cdot \cfrac{1}{x} = 0\)

    当$x $时,\(0 < [x]\cdot \cfrac{1}{x} < 1\)

    根据夹逼定理:\(\lim\limits_{x \to 0^{+}}[x]\cdot \cfrac{1}{x} = 0\)

  • 再证明\(\lim\limits_{x \to 0}x \cdot [\cfrac{1}{x}] = 1\)

    当\(x \ne 0\)时

    \(\cfrac{1}{x}-1 < [\cfrac{1}{x}] \leq \cfrac{1}{x} \Rightarrow 1-x \leq x\cdot [\cfrac{1}{x}]\leq 1\)

    根据夹逼定理:\(\lim\limits_{x \to 0}x \cdot [\cfrac{1}{x}] = 1\)

Example4:求解\(\lim\limits_{n \to ∞}\cfrac{1\cdot 3\cdot 5 \cdot...\cdot (2n-1)}{2\cdot 4\cdot 6\cdot ...\cdot 2n}\)

Solution 记\(a_n = \cfrac{1\cdot 3\cdot 5 \cdot...\cdot (2n-1)}{2\cdot 4\cdot 6\cdot ...\cdot 2n}\)对\(2n\)进行放缩,有\(2n = \cfrac{2n-1+2n+1}{2}\geq \sqrt{(2n-1)(2n+1)}\)

故有\(0 < a_n < \cfrac{1}{\sqrt{2n+1}}\)

根据夹逼定理\(\lim\limits_{n \to ∞}\cfrac{1\cdot 3\cdot 5 \cdot...\cdot (2n-1)}{2\cdot 4\cdot 6\cdot ...\cdot 2n} = 0\)

Example5:计算\(\lim\limits_{n \to ∞}(\cfrac{1}{n^2+1}+\cfrac{1}{n^2 + 2^2} + ... + \cfrac{1}{n^2 + n^2})\)

Solution 记\(a_n = \cfrac{1}{n^2+1}+\cfrac{1}{n^2 + 2^2} + ... + \cfrac{1}{n^2 + n^2}\)

有$ = <a_n< < = $

即\(\cfrac{1}{2n} < a_n < \cfrac{1}{n}\)

根据夹逼定理\(\lim\limits_{n \to ∞}(\cfrac{1}{n^2+1}+\cfrac{1}{n^2 + 2^2} + ... + \cfrac{1}{n^2 + n^2}) = 0\)

取对数

当我们遇到指数型的极限时,可以考虑取对数,即: \[ a^{b} = e^{b\ln a} \] 下面给出几个例子:

Example1:计算\(\lim\limits_{x \to +\infty}x^{1/x}\)

Solution \(\lim\limits_{x \to +\infty}x^{1/x} = \lim\limits_{x \to +\infty}e^{\ln x/x } = e^{\lim\limits_{x \to +\infty}\ln x/x} = e^{0} = 1\)

Example2:计算\(\lim\limits_{x \to 0^{+}}x^x\)

Solution \(\lim\limits_{x \to 0^{+}}x^x = \lim\limits_{x \to 0^{+}}e^{x\ln x} = e^{\lim\limits_{x \to 0^{+}}x\ln x} = e^0 = 1\)

Example3:计算\(\lim\limits_{x \to 0^{+}}x^{\sin x}\)

Solution ${x {+}}x{x} = {x {+}}e{x x} = e^{_{x ^{+}}x x} = e^{1 } =1 $

Example4:计算\(\lim\limits_{x \to +\infty}(\ln x)^{1/x}\)

Solution \(\lim\limits_{x \to +\infty}(\ln x)^{1/x} =\lim\limits_{x \to +\infty} e^{\ln \ln x/ \ln x\cdot \ln x /x} = 1\)

利用两个重要极限

此前我们曾介绍过两个重要极限,他们分别为: \[ \lim_{x \to 0}\cfrac{\sin x}{x} = 1,\lim_{x \to 0}(1+x)^{1/x} = e \] 利用两个重要极限,我们可以计算某些比较复杂代数式的极限,例题如下:

Example1:计算\(\lim\limits_{x \to 0}\cfrac{\sin (ax)}{\sin (bx)}(a,b \ne 0)\)

Solution \(\lim\limits_{x \to 0}\cfrac{\sin (ax)}{\sin (bx)} = \lim\limits_{x \to 0}\cfrac{\sin (ax)}{ax}\cdot \cfrac{bx}{\sin(bx)} \cdot \cfrac{a}{b} = \cfrac{a}{b}\)

Example2:计算\(\lim\limits_{x \to 0}(1+ax)^{b/x}(a,b \ne 0)\)

Solution \(\lim\limits_{x \to 0}(1+ax)^{b/x} = \lim\limits_{x \to 0}(1+ax)^{ab/ax} = e^{ab}\)

Example3:计算\(\lim\limits_{n \to +\infty}\cos(\cfrac{x}{2})\cdot ...\cdot \cos (\cfrac{x}{2^n})(x \ne 0)\)

Solution 记\(f(x) = \cos(\cfrac{x}{2})...\cos (\cfrac{x}{2^n})\)

则\(f(x) = \cfrac{\sin (\cfrac{x}{2^n})}{\sin (\cfrac{x}{2^n})}\cos(\cfrac{x}{2})...\cos (\cfrac{x}{2^n})\)

根据二倍角公式,反复迭代\(n\)次,有

原式\(=\lim\limits_{n \to +\infty} \cfrac{\sin x}{2^n \sin (\cfrac{x}{2^n})} = \lim\limits_{n \to +\infty}\cfrac{\cfrac{x}{2^n}}{\sin \cfrac{x}{2^n}}\cfrac{\sin x}{x} = \cfrac{\sin x}{x}\)

Example4:计算\(\lim\limits_{x \to 0}(1+\cos x)^{3\sec x}\)

Solution 直接计算:原式\(=2^3 = 8\)

Note:注意这个极限并不是重要极限,因为趋势不同.

Example5:计算\(\lim\limits_{x \to 0^{+}}(1+\cfrac{1}{n}+\cfrac{1}{n^2})^n\)与\(\lim\limits_{n \to +\infty}(1+\cfrac{1}{n^2})^n\)

Solution

  • 首先求解\(\lim\limits_{x \to 0^{+}}(1+\cfrac{1}{n}+\cfrac{1}{n^2})^n\)

​ \(\lim\limits_{x \to 0^{+}}(1+\frac{1}{n}+\frac{1}{n^2})^n = \lim\limits_{x \to 0^{+}}[(1+\frac{n+1}{n^2})^{\frac{n^2}{n+1}}]^{\frac{n+1}{n}} = e\)

  • 再计算\(\lim\limits_{n \to +\infty}(1+\cfrac{1}{n^2})^n\)

    \(\lim\limits_{n \to +\infty}(1+\cfrac{1}{n^2})^n = \lim\limits_{n \to +\infty}[(1+\cfrac{1}{n^2})^{n^2}]^{1/n} = 1\)

Example6:计算\(\lim\limits_{x \to 0^{+}}(\cfrac{a_1^x+...+a_n^x}{n})^{1/x}(a_i >0,i = 1,2,...,n)\)

Solution 记\(\Delta = a_1^x-1+a_2^x-1+...+a_n^x-1\)

\(\Rightarrow \lim\limits_{x \to 0^{+}}(\cfrac{a_1^x+...+a_n^x}{n})^{1/x} = \lim\limits_{x \to 0^{+}}[(1+\cfrac{\Delta}{n})^{n/\Delta}]^{1/x\cdot \Delta /n} = \sqrt[n]{a_1a_2...a_n}\)

根式有理化

当所求极限的代数式存在较多根号时,可以考虑使用根式有理化,给出几个例子:

Example1:计算:\(\lim\limits_{x \rightarrow 3} \frac{\sqrt{5 x+1}-\sqrt[3]{20 x+4}}{\sqrt{x+1}-2}\)

Solution \(\lim\limits_{x \rightarrow 3} \frac{\sqrt{5 x+1}-\sqrt[3]{20 x+4}}{\sqrt{x+1}-2} = \lim\limits_{x \rightarrow 3} \frac{\frac{5(x-3)}{\sqrt{5 x+1}+4}-\frac{20(x-3)}{(20 x+4)^{\frac{2}{3}}+4 \sqrt[3]{20 x+4}+16}}{\frac{x-3}{\sqrt{x+1}+2}}=\frac{5}{6}\)

Example2:计算:\(\lim\limits_{x \rightarrow+\infty}(\sqrt{x+\sqrt{x+\sqrt{x}}}-\sqrt{x})\)

Solution \(\lim\limits_{x \rightarrow+\infty}(\sqrt{x+\sqrt{x+\sqrt{x}}}-\sqrt{x}) = \lim\limits_{x \rightarrow+\infty} \frac{\sqrt{x+\sqrt{x}}}{\sqrt{x+\sqrt{x+\sqrt{x}}}+\sqrt{x}}=\frac{1}{2}\)

Example3:计算:\(\lim\limits_{x \to -1}\frac{\frac{2}{1+2 x}+1}{\sqrt[3]{2+x}+x}\)

Solution \(\lim\limits_{x \to -1}\cfrac{\cfrac{2}{1+2 x}+1}{\sqrt[3]{2+x}+x} = \lim\limits_{x \to -1}\cfrac{\cfrac{2(x+1)}{(1+2 x)^{\cfrac{2}{3}}-(1+2 x)^{\cfrac{1}{3}}+1}}{\cfrac{2+x+x^3}{(2+x)^{\cfrac{2}{3}}-x(2+x)^{\cfrac{1}{3}}+x^2}}=\cfrac{1}{2}\)

利用单调有界原理

首先回顾一下极限中的单调有界原理：

Theorem:单调有界数列必有极限

利用这个原理,我们可以计算一些抽象数列的极限,例题如下:

Example1:数列\(\{x_n\}\)满足递推关系:\(x_n = \cfrac{1}{2}(x_{n-1} + \cfrac{a}{x_{n-1}}),a>0,x_n>0\),求证:\(\lim\limits_{n \to \infty}x_n = \sqrt{a}\)

Proof 考虑运用单调有界原理证明

  • 首先证明数列有界,化简递推关系有:

    \(x_{n-1}^2-2x_{n-1}x_{n}+x_{n}^2 = x_{n}^2 - a \Rightarrow x_n^2-a = (x_n-x_{n-1})^2 \geq 0\)

    即\(x_n \geq \sqrt{a}\),数列存在下界.

  • 再证明数列单调,有\(\cfrac{x_n}{x_{n-1}} = \cfrac{1}{2}(1+\cfrac{a}{x_{n-1}^2}) \leq 1 \Rightarrow \{x_n\}\)单调递减,根据单调有界原理,数列\(\{x_n \}\)极限存在

    设\(\lim\limits_{n \to \infty }x_n = A\)则有方程\(A = \cfrac{1}{2}(A+\cfrac{a}{A}) \Rightarrow A = \sqrt{a}\)

Example2:数列\(\{x_n\}\)满足递推关系\(x_{n+1} = 1+\cfrac{x_n}{1+x_n},x_1=1,x_2 = \cfrac{3}{2}\),求$_{n }x_n $

Solution 还是利用单调有界原理

  • 首先证明其单调性:

    $ x_{n_1}-x_n=1+-1- ==A(x_2-x_1)>0x_n $单调递增

  • 再证明其有界性:

    易知\(x_n < 2\),根据单调有界原理,数列\(\{x_n\}\)极限存在,

    设\(\lim\limits_{n \to \infty}x_n = B\),则有方程\(B = 1 + \cfrac{B}{1+B}\),解得\(B = \cfrac{1+\sqrt{5}}{2}\)

Example3:数列\(\{x_n\}\)满足递推关系\(x_1 = \sqrt{2},x_n = \sqrt{2+x_{n-1}}\),求证\(\lim\limits_{n \to \infty}x_n = 2\)

Proof

  • 根据数学归纳法,\(x_n\)单调递增,有上界\(2\)
  • 设\(\lim\limits_{n \to \infty}x_n = A\),有方程\(A = \sqrt{2+A}\),解得\(A = \sqrt{2}\)
