极限与连续

有限闭区间连续函数的性质

最后我们来讨论一下$f(x) \in C[a,b]$的性质吧！

有界性定理

考虑一个函数$f(x)\in C[a,b]$,其满足有界性定理:

Theorem(有界性定理):$f(x)$在$[a,b]$上一定有界.

Note 1:考虑函数$f(x) = \cfrac{1}{x},x\in(0,1)$,易知$f(x)$在$(0,1)$连续,但是$f(x)$在$(0,1)$无界,故有界性定理只在闭区间上成立.

Note 2:考虑函数$f(x) = \begin{cases}\cfrac{1}{x},x\in(0,1] \\ 1,x = 0\end{cases}$,易知$f(x)$$(0,1]$上连续但是无界,故有界性定理要求函数在闭区间上连续.

最值定理

考虑一个函数$f(x) \in C[a,b]$,其满足最值定理:

Theorem(最值定理):$\exists x_1,x_2 \in [a,b]$使得$m \leq f(x_1) \leq f(x_2) \leq M$

Note 1:考虑函数$f(x) = \cfrac{1}{1+e^{1/x}},x \in [-1,1]$,易知$f(x)$在$[-1,1]$上无法取到最值,故最值定理要求函数在闭区间上连续.

Note 2:考虑函数$f(x) = x,x\in (-1,1)$,易知$f(x)$在$(-1,1)$上无法取到最值,故最值定理只在闭区间上成立.

介值定理

考虑一个函数$f(x) \in C[a,b]$,其满足介值定理:

Theorem(介值定理):若$\mu$是介于$f(a),f(b)$之间的任何值,则$\exists \xi \in [a,b]$使得$f(\xi ) = \mu$

特别地,连续函数$f(x)$在$[a,b]$上的最大值,最小值分别设为$M,m$,则对于$\forall \mu \in [m,M],\exists \xi \in [a,b]$使得$f(\xi) = \mu$.

零点存在性定理

考虑函数$f(x) \in C[a,b]$,其满足零点存在性定理:

Theorem(零点存在性定理):若函数$f(x)$满足$f(a)\cdot f(b) < 0$,则$\exists \xi \in (a,b)$,使得$f(\xi) = 0$

利用零点存在性定理,我们往往可以判断一个连续函数的零点个数及其范围,给出几个例子:

Example 1:求方程$x^3+4x^2-3x-1=0$解的个数.

Solution 记函数$f(x) =x^3+4x^2-3x-1$,

易知$f(-10) = -5 < 0,f(-1) = 5 > 0 ,f(0) = -1,f(1) = 1> 0$

故根据零点存在性定理,方程有两个解

Example 2:求方程$4-x = 2^x$在$[1,2]$的解的个数

Solution 记$f(x) = 4-x-2^x,x\in[1,2]$

易知$f(1) = 1,f(2)=-2$,且函数在$[1,2]$上单调递减,则有一个实根

例题

关于本部分,最后给出几个例题:

Example 1:$\lim\limits_{x\to -1}\cfrac{ax^2-x-3}{x+1} = b$,求$a,b$的值.

Solution $I = -2a-1 = b$

由极限的存在性可知$a = 2,b = -5$

Example 2:已知$\lim\limits_{n\to \infty}(3n-\sqrt{an^2+bn+1}) = 2$,求$a,b$的值.

Solution $I = \lim\limits_{n\to \infty}\cfrac{(9-a)n^2-bn-1}{3n+\sqrt{an^2+bn+1}} = 2$

由极限的存在性可知$a= 9,b=-12$

Example 3:已知函数$f(x) = \lim\limits_{n\to\infty}\cfrac{x^{2n-1}+ax^2+bx}{x^{2n}+1}$在$R$上连续,求$a,b$的值.

Solution 分情况讨论,可知函数$f(x)$的表达形式为:

$f(x)=\begin{cases} \cfrac{1}{x},x<-1 \\\cfrac{1}{2}(a-b-1),x = -1\\ ax^2+bx,-11\end{cases}$

根据连续的定义,我们可以求得$a=0,b=1$
