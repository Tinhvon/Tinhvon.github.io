微积分-Chapter1-极限与连续（Part 1）

极限与连续（Part 1）

极限

言简意赅,我们首先给出在某点处函数极限的定义:

Definition:设$y=f(x)$在$N_{0}(x_0,\delta)$有定义,则对于$\forall \varepsilon > 0,\exists \delta >0$,当$x \in N_0(x_0,\delta)$时,有$|f(x)-A| < \varepsilon$,则称$f(x)$在$x_0$处的极限为$A$,记为$\lim\limits_{x \to x_0} f(x) = A$

exp1:证明: $\lim\limits_{x \to x_0}x = x_0$

Proof:对于$\forall \varepsilon >0$,取$\delta = \varepsilon$.,当$0<|x-x_0|<\delta$时,有:

$|x-x_0|<\varepsilon$

$\Rightarrow \lim\limits_{x \to x_0}x = x_0$

exp2:证明$\lim\limits_{x \to x_0}\cfrac{1}{x} = \cfrac{1}{x_0}$

Proof:对于$\forall \varepsilon >0,|x-x_0| < \cfrac{|x_0|}{2} \Rightarrow |x| > \cfrac{x_0}{2} $

取$\delta = \min \{ \cfrac{|x_0|}{2},\cfrac{|x_0|^2}{2}\varepsilon\}$

当$x \in N_{0}(x_0,\delta) \Rightarrow |\cfrac{1}{x}-\cfrac{1}{x_0}| < \varepsilon$

$\Rightarrow \lim\limits_{x \to x_0}\cfrac{1}{x} = \cfrac{1}{x_0}$

单侧极限

给出普遍的函数极限定义后,我们可更进一步,给出单侧极限的定义:

Definition:设$y=f(x)$在$N_{0}^{+}(x_0,\delta)$有定义,则对于$\forall \varepsilon > 0,\exists \delta >0$,当$x \in N_0^{+}(x_0,\delta)$时,有$|f(x)-A| < \varepsilon$,则称$f(x)$在$x_0$处的右极限为$A$,记为$\lim\limits_{x \to x_0^{+}} f(x) = A$

Definition:设$y=f(x)$在$N_{0}^{-}(x_0,\delta)$有定义,则对于$\forall \varepsilon > 0,\exists \delta >0$,当$x \in N_0^{-}(x_0,\delta)$时,有$|f(x)-A| < \varepsilon$,则称$f(x)$在$x_0$处的左极限为$A$,记为$\lim\limits_{x \to x_0^{-}} f(x) = A$

给出单侧极限的定义后,我们可以得到其推论:

$\lim\limits_{x\to x_0}f(x)=A \Leftrightarrow \lim\limits_{x \to x_0^{+}} f(x)=\lim\limits_{x \to x_0^{-}} f(x) = A$

exp3:已知函数:

证明:$\lim\limits_{x \to 0}f(x)$不存在

Proof:$\lim\limits_{x\to 0^{+}}=1,\lim\limits_{x\to 0^{-}}=-1$两者不相同,故极限不存在.

无穷远点的极限

现在我们讨论无穷远点的极限的$\varepsilon-\delta$语言描述:

Definition:设$y=f(x)$在$\{x| |x|>M\}$有定义,则对于$\forall \varepsilon > 0,\exists x_{0} >0$,当$|x|>x_{0}$时,有$|f(x)-A| < \varepsilon$,则称$f(x)$在$∞$处的极限为$A$,记为$\lim\limits_{x \to ∞} f(x) = A$

在定义无穷远点极限的基础上,我们也可将单侧极限的定义引入进来,定义如下:

Definition:设$y=f(x)$在$\{x| |x|>M\}$有定义,则对于$\forall \varepsilon > 0,\exists x_{0} >0$,当$x>x_{0}$时,有$|f(x)-A| < \varepsilon$,则称$f(x)$在$+∞$处的极限为$A$,记为$\lim\limits_{x \to +∞} f(x) = A$

Definition:设$y=f(x)$在$\{x| |x|>M\}$有定义,则对于$\forall \varepsilon > 0,\exists x_{0} >0$,当$x<-x_{0}$时,有$|f(x)-A| < \varepsilon$,则称$f(x)$在$-∞$处的极限为$A$,记为$\lim\limits_{x \to -∞} f(x) = A$

exp4:证明$\lim\limits_{x \to ∞}\cfrac{1}{x} = 0$

Proof:对$\forall \varepsilon > 0$,取$x_{0} = \cfrac{1}{\varepsilon}+ 1$,当$|x|>x_0$时:

$|\cfrac{1}{x} - 0| < \varepsilon \Rightarrow \lim\limits_{x \to ∞}\cfrac{1}{x} = 0$

极限为无穷大

下面给出极限为无穷大的$\varepsilon-\delta$语言描述

Definition:设$y=f(x)$在$N_{0}(x_0,\delta)$有定义,且对于$\forall M > 0,\exists \delta >0$,当$x \in N_{0}(x_0,\delta)$时,有$|f(x)|>M$,则称$f(x)$在$x_0$处的极限为$∞$(极限不存在),记为$\lim\limits_{x \to x_0} f(x) = ∞$

exp5:$\lim\limits _{x \to 1} \cfrac{x}{(x-1)^{2}} = ∞$

数列的极限

此前我们讨论的均为函数的极限,数列是特殊的函数,其极限的$\varepsilon - \delta $语言描述如下:

Definition:设数列$\{a_n\}$,对于$\forall \varepsilon > 0$,当$n > N$时,$|a_n - A| < \varepsilon$,则称数列的极限为$A$,记为$\lim\limits _{n \to ∞} a_n = A$

exp6:证明$\lim\limits_{n \to ∞} \sqrt[n]a = 1(a>1)$

Proof:$\sqrt[n]a = \sqrt[n]{a \cdot 1 \cdot1 … \cdot 1}(n-1个1)$

由均值不等式:$\sqrt[n]a = \sqrt[n]{a \cdot 1 \cdot1 … \cdot 1} \leq \cfrac{a + n - 1}{n} = 1 + \cfrac{a-1}{n}$

故有$0 <\sqrt[n]{a}-1 < \cfrac{a-1}{n} $

对于$\forall \varepsilon > 0$,取$N = [\cfrac{a-1}{\varepsilon}]+1$,当$n>N$时有:

$|\sqrt[n]a - 1 | < \varepsilon \Rightarrow \lim\limits_{n \to ∞} \sqrt[n]a = 1$

exp7:证明$\lim\limits _{n \to ∞} q ^{n}= 0(|q| < 1 )$

Proof:分两种情况讨论:

①:$q = 0$ ,显然成立

②:$0 < |q| < 1$时,记$h = \cfrac{1}{|q|} - 1$

则有$|q^n| = \cfrac{1}{(1+h)^{n}} \leq \cfrac{1}{1+nh} \leq \cfrac{1}{nh}$

对于$\forall \varepsilon > 0$,取$N = [\cfrac{1}{\varepsilon h}]+ 1$,当$n > N $时,有:

$|q^n - 0| < \varepsilon \Rightarrow \lim\limits _{n \to ∞} q ^{n}= 0(|q| < 1 )$

exp8: 证明$\lim\limits _{n \to ∞} \sqrt[n]{n} = 1$

Proof:记$\sqrt[n]{n} = a_n,n>1 \Rightarrow a_n > 1$

记$a_n = 1 + \alpha_n ,\alpha > 0$

故$n = (1 + \alpha_n)^{n} \geq \cfrac{n}{2}(n-1)\alpha_{n}^{2}\Rightarrow 0< \alpha_n \leq \sqrt{\cfrac{2}{n-1}}$

对于$\forall \varepsilon > 0$,取$N = [\cfrac{2}{\varepsilon ^2} + 1] + 1$,当$n > N $时,有:

$|\sqrt[n]{n} - 1| < \varepsilon \Rightarrow \lim\limits _{n \to ∞} \sqrt[n]{n} = 1$

极限的性质

上文中我们介绍了各种各样的极限,现在我们讨论一下他的性质:

运算法则

首先讨论极限的运算:不妨假设$\lim\limits_{x \to x_0}f(x) = A,\lim\limits_{x \to x_0}g(x) = B$,则其满足:

①:$\lim\limits_{x \to x_0}[\alpha f(x)+\beta g(x)] = \alpha A + \beta B$.

②:$\lim\limits_{x \to x_0}f(x)g(x) = AB$.

③:$\lim\limits_{x \to x_0}\cfrac{f(x)}{g(x)} = \cfrac{A}{B}(B \ne 0)$.

对于复合函数的极限,我们做如下定义:

Definition:假设$f(u)$在$N_{0}(u_0,\delta_1)$有定义,且$\lim\limits _{u \to u_0} = A$,$u = g(x)$在$N_{0}(x_0,\delta_2)$有定义,$\lim\limits_{x \to x_0}g(x) = u_0$,则$\lim\limits_{x \to x_0 }f[g(x)] = A$

唯一性

极限的唯一性定理如下:

Theorem:若函数的极限存在,则其一定唯一.

证明如下,考虑使用反证法:

Proof:不妨假设:$\lim\limits_{x \to x_0}f(x) = A,\lim\limits_{x \to x_0}f(x) = B (A < B)$

取$\varepsilon_0 = \cfrac{B-A}{2} > 0$

由$\lim\limits_{x \to x_0}f(x) = A$

$\Rightarrow \exists \delta_1 > 0$当$x \in N(x_0,\delta_1)$时,$|f(x) - A| < \cfrac{B-A}{2},f(x) < \cfrac{A+B}{2}$

由$\lim\limits_{x \to x_0}f(x) = B$

$\Rightarrow \exists \delta_2 > 0$当$x \in N(x_0,\delta_2)$时,$|f(x) - B| < \cfrac{B-A}{2},f(x) > \cfrac{A+B}{2}$

产生矛盾,故极限唯一性定理得证.

局部有界性

极限局部有界性的表述如下:

Theorem:若$\lim\limits_{x \to x_0}f(x) = A$,则$\exists N(x_0,\delta)$,当$x\in N(x_0,\delta)$时,由$|f(x)| < M$

证明如下:

Proof:取$\varepsilon_0=1$,由$\lim\limits_{x \to x_0}f(x) = A$

$\exists \delta > 0$,当$x\in N(x_0,\delta),|f(x)-A| < 1 \Rightarrow A-1 < f(x) < A + 1$

记$M = |A-1|+|A+1|+1 \Rightarrow |f(x)| < M$

局部保号性

极限局部保号性的表述如下:

Theorem:若$\lim\limits_{x \to x_0}f(x) = A > 0$,则$\forall r \in (0,A),\exists \delta > 0$,当$x \in N_{0}(x_0,\delta)$时 有$f(x) > r > 0$

证明如下:

Proof:取$\varepsilon_0 = A - r$,由$\lim\limits_{x \to x_0}f(x) = A $

$\exists \delta > 0$,当$x \in N_{0} (x_0,\delta )$时,有:

$|f(x)-A| < A - r \Rightarrow f(x) > r$

极限不等式

直接给出极限不等式的定理,这里不再证明

Theorem:若$\lim\limits_{x \to x_0}f(x) = A, \lim\limits_{x \to x_0}g(x) = B$,若$f(x) \leq g(x) ,x \in N_{0}(x,\delta)$,则$A \leq B$

夹逼定理

夹逼定理是极限论中很重要的一个定理,根据此可以得出一个重要极限,先给出定理的具体表述:

夹逼定理(Squeeze Theorem):若$\lim\limits_{x \to x_0}f(x) = \lim\limits_{x \to x_0}g(x) = A$,则$\exists \delta > 0$,当$x \in N(x_0,\delta)$时,有$f(x) \leq h(x) \leq g(x) $,则$\lim\limits_{x \to x_0}h(x) = A$

证明如下:

Proof:$\forall \varepsilon > 0$,由$\lim\limits_{x \to x_0}f(x) = A$,$\exists \delta_1 > 0,x \in N_{0}(x_0,\delta_1),A-\varepsilon < f(x) < A + \varepsilon$

由$\lim\limits_{x \to x_0}g(x) = A$,$\exists \delta_2 > 0,x \in N_{0}(x_0,\delta_1),A-\varepsilon < g(x) < A + \varepsilon$

取$\delta = \min \{\delta_1,\delta_2 \}$,当$x \in N(x_0,\delta)$时,$A-\varepsilon < f(x) \leq h(x) < g(x) < A + \varepsilon$

$\Rightarrow \lim\limits_{x \to x_0}h(x) = A$

通过夹逼定理,我们可以得到一个重要极限:

证明如下:

Proof:当$0<x<\cfrac{\pi}{2}$时,有$\sin x < x < \tan x$

即:$\cfrac{1}{2}\sin x <\cfrac{1}{2}x <\cfrac{1}{2}\tan x \Rightarrow \cos x < \cfrac{\sin x}{x} < 1$

又有$1 - \cos x = 2\sin^2 \cfrac{x}{2} \leq \cfrac{1}{2}x^2 \Rightarrow 1 - \cfrac{1}{2}x^2 \leq \cos x \leq 1$

$\Rightarrow \lim\limits_{x \to 0^{+}}\cfrac{\sin x}{x} = 1$,此时令$x = -y$

有:$ \lim\limits_{y \to 0^{-}}\cfrac{\sin (-y)}{-y} = 1$

综上:$\lim\limits_{x \to 0}\cfrac{\sin x}{x} = 1$

单调有界原理

另一个很重要的定理是单调有界原理,叙述如下:

Theorem:单调有界数列必有极限

证明如下:

Proof:不妨假设,数列$\{a_n\}$单调递增有上界,记$\sup \{a_n\} = a$

则对于$\forall \varepsilon > 0,\exists N > 0,a-\varepsilon < a_{N}$

由于$\{a_n\}$单调递增,当$n>N$时,有$a - \varepsilon < a_{N} \leq a_{n} < a < a + \varepsilon$

即$\lim\limits_{n \to ∞ }a_n = a$

根据单调有界原理,我们可以的到另一个重要极限:

首先证明极限存在:

Proof:记数列$\{x_n\} =(1+\cfrac{1}{n})^{n}$

$\Rightarrow x_n = 1 \cdot (1 + \cfrac{1}{n})\cdot (1+\cfrac{1}{n})…\cdot (1+\cfrac{1}{n}) \leq (\cfrac{n+2}{n+1})^{n+1} = (1 + \cfrac{1}{n+1})^{n+1} = x_{n+1} $可知数列$x_n$单调

又$x_{n} = (1+\cfrac{1}{n})^{n} = 1 + C_{n}^{1}\cfrac{1}{n}+…+C_{n}^{n}\cfrac{1}{n^{n}} \leq 1 + 1 + \cfrac{1}{2} + … + \cfrac{1}{2^{n+1}} < 3$可知$x_n$存在上界

根据单调有界原理,$\lim\limits_{n \to +∞} (1+\cfrac{1}{n})^n$存在,设其为$e$,易知$2<e<3$

现在将数列情况推广至函数情况

当$x \geq 1$时,$(1+\cfrac{1}{1+[x]})^{[x]} < (1+\cfrac{1}{x})^{x} < (1 + \cfrac{1}{[x]})^{[x]+1}$

且$\lim\limits_{n \to +∞}(1+\cfrac{1}{1+[x]})^{[x]} = \lim\limits_{n \to +∞} (1 + \cfrac{1}{[x]})^{[x]+1} = e$

根据夹逼定理:$\lim\limits_{x \to +∞} (1+\cfrac{1}{x})^x = e$

再令$x = -y-1$,易证:$\lim\limits_{y \to -∞} (1+\cfrac{1}{y})^y = e$

综上所述:$\lim\limits_{x \to ∞} (1+\cfrac{1}{x})^x = e$

同理,有一个容易犯错误的地方,下式极限也为e,但是x的趋势不同

海涅定理

最后讨论一下数学分析中常用的一个定理——海涅定理(Heine Theorem),定理叙述如下:

Heine Theorem:对于函数$f(x)$,$\lim\limits_{x \to x_0}f(x) = A \Leftrightarrow \forall x_{n} (x_n \ne x_0),x_n \to x_0,\lim\limits_{n \to +∞}f(x_{n}) = A \Rightarrow \lim\limits_{x \to x_0}f(x) = A,x_n(x_n \ne x_0) \to x_0$

证明如下:

Proof:对于$\forall \varepsilon > 0,\exists \delta > 0$,当$x \in N_{0}(x_0,\delta)$时,$|f(x) - A| < \varepsilon $

而对$\delta > 0$,由$\lim\limits_{n \to +∞}x_n= x_0,\exists N > 0$当$n > N$时,有$|x_n -x_0| < \delta $

故当$n > N$时,有$|f(x_n) - A| < \varepsilon $,即$\lim\limits_{x \to x_0}f(x) = A$

用反证法也可证明,这里不再讨论.
