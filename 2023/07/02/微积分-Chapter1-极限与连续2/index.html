极限与连续（Part 2）

极限的计算

在讨论完极限的定义和各种性质后,我们主要关心的是如何计算极限,下面介绍一些计算极限的方法:

夹逼定理法

exp1:计算$\lim\limits_{n \to ∞}\sqrt[n]{1^n+2^n+…+2022^n}$

Solution:考虑数列$a_n = \sqrt[n]{1^n+2^n+…+2022^n}$

则有$2022^n \leq 1^n+2^n+…+2022^n \leq 2022\times 2022^n$

即有$2022 \leq a_n \leq 2022\sqrt[n]{2022}$

又$\lim\limits_{n \to ∞}2022 = \lim\limits_{n \to ∞}2022\sqrt[n]{2022} = 2022$

根据夹逼定理,有$\lim\limits_{n \to ∞}\sqrt[n]{1^n+2^n+…+2022^n} = 2022$

exp2:证明:$\lim\limits_{n \to ∞}\cfrac{n^{k}}{a^{n}}(a > 1) = 0,\lim\limits_{n \to ∞}\cfrac{\ln n}{n}= 0,\lim\limits_{n \to ∞}\cfrac{c^n}{n!}(c > 1) = 0$（数列增长速度的比较）

Proof:

  • 首先证明$\lim\limits_{n \to ∞}\cfrac{n^{k}}{a^{n}}(a > 1) = 0$

    记$a = 1 + \alpha (\alpha > 0) \Rightarrow a^{n} = (1 + \alpha)^{n} > \cfrac{n(n-1)…(n-k)}{(k+1)!}\alpha^{k+1} = An(n-1)…(n-k)$

    有$0 < \cfrac{n^{k}}{a^{n}} < \cfrac{n^k}{An(n-1)…(n-k)}$

    根据夹逼定理:$\lim\limits_{n \to ∞}\cfrac{n^{k}}{a^{n}}(a > 1) = 0$

  • 再证明$\lim\limits_{n \to ∞}\cfrac{\ln n}{n}= 0$

    记$\ln n = t,n = e^t$

    则根据夹逼定理显然有$\lim\limits_{n \to ∞}\cfrac{\ln n}{n} = \lim\limits_{t \to ∞}\cfrac{t}{e^t} = 0$

    再证明$\lim\limits_{n \to ∞}\cfrac{\ln n}{n}= 0$

  • 最后证明$\lim\limits_{n \to ∞}\cfrac{c^n}{n!}(c > 1) = 0$

    记$[c] = k$

    有$0 < \cfrac{c^n}{n!} = \cfrac{c\cdot c\cdot …\cdot c}{1\cdot 2…\cdot k}\cdot \cfrac{c\cdot … \cdot c\cdot c}{(k+1) …(n-1)n}$

    化简有$0 < \cfrac{c^n}{n!} < \cfrac{Ac}{n} $

    根据夹逼定理$\lim\limits_{n \to ∞}\cfrac{c^n}{n!}(c > 1) = 0$

Note:当$n\to ∞$时,数列增长速度从小到大排列为$\ln n < n^k < a^n < n! < n^n(a > 1)$

exp3:证明以下极限$\lim\limits_{x \to 0^{+}}[x]\cdot \cfrac{1}{x} = 0,\lim\limits_{x \to 0}x \cdot [\cfrac{1}{x}] = 1$

Proof:

  • 首先证明$\lim\limits_{x \to 0^{+}}[x]\cdot \cfrac{1}{x} = 0$

    当$x \ne 0 $时,$0 < [x]\cdot \cfrac{1}{x} < 1$

    根据夹逼定理:$\lim\limits_{x \to 0^{+}}[x]\cdot \cfrac{1}{x} = 0$

  • 再证明$\lim\limits_{x \to 0}x \cdot [\cfrac{1}{x}] = 1$

    当$x \ne 0$时

    $\cfrac{1}{x}-1 < [\cfrac{1}{x}] \leq \cfrac{1}{x} \Rightarrow 1-x \leq x\cdot [\cfrac{1}{x}]\leq 1$

    根据夹逼定理:$\lim\limits_{x \to 0}x \cdot [\cfrac{1}{x}] = 1$

exp4:求解$\lim\limits_{n \to ∞}\cfrac{1\cdot 3\cdot 5 \cdot…\cdot (2n-1)}{2\cdot 4\cdot 6\cdot …\cdot 2n}$

Solution:记$a_n = \cfrac{1\cdot 3\cdot 5 \cdot…\cdot (2n-1)}{2\cdot 4\cdot 6\cdot …\cdot 2n}$对$2n$进行放缩,有$2n = \cfrac{2n-1+2n+1}{2}\geq \sqrt{(2n-1)(2n+1)}$

故有$0 < a_n < \cfrac{1}{\sqrt{2n+1}}$

根据夹逼定理$\lim\limits_{n \to ∞}\cfrac{1\cdot 3\cdot 5 \cdot…\cdot (2n-1)}{2\cdot 4\cdot 6\cdot …\cdot 2n} = 0$

exp5:计算$\lim\limits_{n \to ∞}(\cfrac{1}{n^2+1}+\cfrac{1}{n^2 + 2^2} + … + \cfrac{1}{n^2 + n^2})$

Solution:记$a_n = \cfrac{1}{n^2+1}+\cfrac{1}{n^2 + 2^2} + … + \cfrac{1}{n^2 + n^2}$

有$\cfrac{n}{n^2 + n^2} = \cfrac{1}{2n}<a_n<\cfrac{n}{n^2+1} < \cfrac{n}{n^2} = \cfrac{1}{n} $

即$\cfrac{1}{2n} < a_n < \cfrac{1}{n}$

根据夹逼定理$\lim\limits_{n \to ∞}(\cfrac{1}{n^2+1}+\cfrac{1}{n^2 + 2^2} + … + \cfrac{1}{n^2 + n^2}) = 0$

取对数

当我们遇到指数型的极限时,可以考虑取对数,即:

下面给出几个例子:

exp1:计算$\lim\limits_{x \to +\infty}x^{1/x}$

Solution:$\lim\limits_{x \to +\infty}x^{1/x} = \lim\limits_{x \to +\infty}e^{\ln x/x } = e^{\lim\limits_{x \to +\infty}\ln x/x} = e^{0} = 1$

exp2:计算$\lim\limits_{x \to 0^{+}}x^x$

Solution:$\lim\limits_{x \to 0^{+}}x^x = \lim\limits_{x \to 0^{+}}e^{x\ln x} = e^{\lim\limits_{x \to 0^{+}}x\ln x} = e^0 = 1$

exp3:计算$\lim\limits_{x \to 0^{+}}x^{\sin x}$

Solution:$\lim\limits_{x \to 0^{+}}x^{\sin x} = \lim\limits_{x \to 0^{+}}e^{\sin x \ln x} = e^{\lim\limits_{x \to 0^{+}}\cfrac{\sin x}{x}\cdot x \ln x} = e^{1 \times 0} =1 $

exp4:计算$\lim\limits_{x \to +\infty}(\ln x)^{1/x}$

Solution:$\lim\limits_{x \to +\infty}(\ln x)^{1/x} =\lim\limits_{x \to +\infty} e^{\ln \ln x/ \ln x\cdot \ln x /x} = 1$

利用两个重要极限

此前我们曾介绍过两个重要极限,他们分别为:

利用两个重要极限,我们可以计算某些比较复杂代数式的极限,例题如下:

exp1:计算$\lim\limits_{x \to 0}\cfrac{\sin (ax)}{\sin (bx)}(a,b \ne 0)$

Solution:$\lim\limits_{x \to 0}\cfrac{\sin (ax)}{\sin (bx)} = \lim\limits_{x \to 0}\cfrac{\sin (ax)}{ax}\cdot \cfrac{bx}{\sin(bx)} \cdot \cfrac{a}{b} = \cfrac{a}{b}$

exp2:计算$\lim\limits_{x \to 0}(1+ax)^{b/x}(a,b \ne 0)$

Solution:$\lim\limits_{x \to 0}(1+ax)^{b/x} = \lim\limits_{x \to 0}(1+ax)^{ab/ax} = e^{ab}$

exp3:计算$\lim\limits_{n \to +\infty}\cos(\cfrac{x}{2})\cdot …\cdot \cos (\cfrac{x}{2^n})(x \ne 0)$

Solution:记$f(x) = \cos(\cfrac{x}{2})…\cos (\cfrac{x}{2^n})$

则$f(x) = \cfrac{\sin (\cfrac{x}{2^n})}{\sin (\cfrac{x}{2^n})}\cos(\cfrac{x}{2})…\cos (\cfrac{x}{2^n})$

根据二倍角公式,反复迭代$n$次,有

原式$=\lim\limits_{n \to +\infty} \cfrac{\sin x}{2^n \sin (\cfrac{x}{2^n})} = \lim\limits_{n \to +\infty}\cfrac{\cfrac{x}{2^n}}{\sin \cfrac{x}{2^n}}\cfrac{\sin x}{x} = \cfrac{\sin x}{x}$

exp4:计算$\lim\limits_{x \to 0}(1+\cos x)^{3\sec x}$

Solution:直接计算:原式$=2^3 = 8$

Note:注意这个极限并不是重要极限,因为趋势不同.

exp5:计算$\lim\limits_{x \to 0^{+}}(1+\cfrac{1}{n}+\cfrac{1}{n^2})^n$与$\lim\limits_{n \to +\infty}(1+\cfrac{1}{n^2})^n$

Solution:

  • 首先求解$\lim\limits_{x \to 0^{+}}(1+\cfrac{1}{n}+\cfrac{1}{n^2})^n$

​ $\lim\limits_{x \to 0^{+}}(1+\frac{1}{n}+\frac{1}{n^2})^n = \lim\limits_{x \to 0^{+}}[(1+\frac{n+1}{n^2})^{\frac{n^2}{n+1}}]^{\frac{n+1}{n}} = e$

  • 再计算$\lim\limits_{n \to +\infty}(1+\cfrac{1}{n^2})^n$

    $\lim\limits_{n \to +\infty}(1+\cfrac{1}{n^2})^n = \lim\limits_{n \to +\infty}[(1+\cfrac{1}{n^2})^{n^2}]^{1/n} = 1$

exp6:计算$\lim\limits_{x \to 0^{+}}(\cfrac{a_1^x+…+a_n^x}{n})^{1/x}(a_i >0,i = 1,2,…,n)$

Solution:记$\Delta = a_1^x-1+a_2^x-1+…+a_n^x-1$

$\Rightarrow \lim\limits_{x \to 0^{+}}(\cfrac{a_1^x+…+a_n^x}{n})^{1/x} = \lim\limits_{x \to 0^{+}}[(1+\cfrac{\Delta}{n})^{n/\Delta}]^{1/x\cdot \Delta /n} = \sqrt[n]{a_1a_2…a_n}$

根式有理化

当所求极限的代数式存在较多根号时,可以考虑使用根式有理化,给出几个例子:

exp1:计算:$\lim\limits_{x \rightarrow 3} \frac{\sqrt{5 x+1}-\sqrt[3]{20 x+4}}{\sqrt{x+1}-2}$

Solution:$\lim\limits_{x \rightarrow 3} \frac{\sqrt{5 x+1}-\sqrt[3]{20 x+4}}{\sqrt{x+1}-2} = \lim\limits_{x \rightarrow 3} \frac{\frac{5(x-3)}{\sqrt{5 x+1}+4}-\frac{20(x-3)}{(20 x+4)^{\frac{2}{3}}+4 \sqrt[3]{20 x+4}+16}}{\frac{x-3}{\sqrt{x+1}+2}}=\frac{5}{6}$

exp2:计算:$\lim\limits_{x \rightarrow+\infty}(\sqrt{x+\sqrt{x+\sqrt{x}}}-\sqrt{x})$

Solution:$\lim\limits_{x \rightarrow+\infty}(\sqrt{x+\sqrt{x+\sqrt{x}}}-\sqrt{x}) = \lim\limits_{x \rightarrow+\infty} \frac{\sqrt{x+\sqrt{x}}}{\sqrt{x+\sqrt{x+\sqrt{x}}}+\sqrt{x}}=\frac{1}{2}$

exp3:计算:$\lim\limits_{x \to -1}\frac{\frac{2}{1+2 x}+1}{\sqrt[3]{2+x}+x}$

Solution:$\lim\limits_{x \to -1}\cfrac{\cfrac{2}{1+2 x}+1}{\sqrt[3]{2+x}+x} = \lim\limits_{x \to -1}\cfrac{\cfrac{2(x+1)}{(1+2 x)^{\cfrac{2}{3}}-(1+2 x)^{\cfrac{1}{3}}+1}}{\cfrac{2+x+x^3}{(2+x)^{\cfrac{2}{3}}-x(2+x)^{\cfrac{1}{3}}+x^2}}=\cfrac{1}{2}$

exp4:计算:$ \lim\limits_{n \rightarrow \infty} \sin \left(\pi \sqrt{n^2+1}\right)$

Solution:$ \lim\limits_{n \rightarrow \infty}\sin \left(\pi \sqrt{n^2+1}\right) =\lim\limits_{n \to +\infty}(-1)^n \sin \left(\pi \sqrt{n^2+1}-n \pi\right)=\lim\limits_{n \rightarrow \infty}(-1)^{n} \sin \frac{\pi}{\sqrt{n^2+1}+n}=0 \\
$

约分法

一般遇到两多项式商类型的极限,即:

其中$P(x),Q(x)$是多项式,一个很常用的思路是因式分解,即约分法,下面给出几个例子:

exp1:计算$\lim\limits_{x \rightarrow 1} \frac{x^m-1}{x^n-1} \quad\left(m, n \in N^{+}\right)$

Solution:$\lim\limits_{x \rightarrow 1} \cfrac{x^m-1}{x^n-1}=\lim\limits_{x \rightarrow 1} \cfrac{(x-1)\left(x^{m-1}+x^n+\cdots+1\right)}{(x-1)\left(x^{n-1}+x^{+2}+\cdots+1\right)}=\cfrac{m}{n}
$

exp2:计算$\lim\limits_{x \to 0}\cfrac{\sqrt[n]{1+x}-1}{x}
$

Solution:记$\sqrt[n]{1+x}-1 = t$

故$\lim\limits_{x \to 0}\cfrac{\sqrt[n]{1+x}-1}{x} = \lim\limits_{t \to 1}\cfrac{t-1}{t^n-1} = \cfrac{1}{n}$

利用单调有界原理

首先回顾一下极限中的单调有界原理

Theorem:单调有界数列必有极限

利用这个原理,我们可以计算一些抽象数列的极限,例题如下:

exp1:数列$\{x_n\}$满足递推关系:$x_n = \cfrac{1}{2}(x_{n-1} + \cfrac{a}{x_{n-1}}),a>0,x_n>0$,求证:$\lim\limits_{n \to \infty}x_n = \sqrt{a}$

Proof:考虑运用单调有界原理证明

  • 首先证明数列有界,化简递推关系有:

    $x_{n-1}^2-2x_{n-1}x_{n}+x_{n}^2 = x_{n}^2 - a \Rightarrow x_n^2-a = (x_n-x_{n-1})^2 \geq 0$

    即$x_n \geq \sqrt{a}$,数列存在下界.

  • 再证明数列单调,有$\cfrac{x_n}{x_{n-1}} = \cfrac{1}{2}(1+\cfrac{a}{x_{n-1}^2}) \leq 1 \Rightarrow \{x_n\}$单调递减,根据单调有界原理,数列$\{x_n \}$极限存在

    设$\lim\limits_{n \to \infty }x_n = A$则有方程$A = \cfrac{1}{2}(A+\cfrac{a}{A}) \Rightarrow A = \sqrt{a}$

exp2:数列$\{x_n\}$满足递推关系$x_{n+1} = 1+\cfrac{x_n}{1+x_n},x_1=1,x_2 = \cfrac{3}{2}$,求$\lim\limits_{n \to \infty}x_n $

Solution:还是利用单调有界原理

  • 首先证明其单调性:

    $ x_{n_1}-x_n=1+\cfrac{x_n}{1+x_n}-1-\cfrac{x_{n-1}}{1+x_{-1}}
    =\cfrac{x_{n-1}-x_{n-1}}{\left(1+x_n\right)\left(1+x_{n-1}\right)}=A\left(x_2-x_1\right)>0\Rightarrow x_n
    $单调递增

  • 再证明其有界性:

    易知$x_n < 2$,根据单调有界原理,数列$\{x_n\}$极限存在,

    设$\lim\limits_{n \to \infty}x_n = B$,则有方程$B = 1 + \cfrac{B}{1+B}$,解得$B = \cfrac{1+\sqrt{5}}{2}$

exp3:数列$\{x_n\}$满足递推关系$x_1 = \sqrt{2},x_n = \sqrt{2+x_{n-1}}$,求证$\lim\limits_{n \to \infty}x_n = 2$

Proof:

  • 根据数学归纳法,$x_n$单调递增,有上界$2$
  • 设$\lim\limits_{n \to \infty}x_n = A$,有方程$A = \sqrt{2+A}$,解得$A = \sqrt{2}$

无穷小量

首先给出无穷小量的定义

Definition:若$x\to x_0$时,$\alpha(x) \to 0$,则称$\alpha (x) $为$x\to x_0$的无穷小量.

数学语言表述为:$\lim\limits_{x \to x_0}f(x) = A \Leftrightarrow f(x) = A +\alpha ,\lim\limits_{x \to x_0} \alpha =0$

相关推论

给出无穷小的定义后,我们对无穷小做进一步讨论

Note1:

  • 有限个无穷小之和为无穷小
  • 有限个无穷小之积为无穷小
  • 有界量与无穷小之积也为无穷小
  • 无穷个无穷小之和不一定为无穷小,如$x_n = \cfrac{1}{n},n\to \infty$
  • 无穷个无穷小之积不一定为无穷小

以上证明略,由Note1的第三点可以得知$\lim\limits_{x \to 0}x\sin(\cfrac{1}{x}) = 0$

讨论无穷小和与积的性质后,我们暂未搞懂应当怎样描述无穷小的大小,先给出推论:

Note2:

  • $\lim\limits_{x \to x_0}\cfrac{\alpha(x)}{\beta(x)}= 0$,称$\alpha(x)$是$\beta(x)$的高阶无穷小,记作$\alpha(x) = o(\beta(x))$
  • $\lim\limits_{x \to x_0}\cfrac{\alpha(x)}{\beta(x)}= C \ne 0$,称$\alpha(x)$是$\beta(x)$的同阶无穷小,记作$\alpha(x) \textasciitilde C\beta(x)$,若$C = 1$,称$\alpha(x)$是$\beta(x)$的等价无穷小,记作$\alpha(x) \textasciitilde \beta(x),x\to x_0$

在掌握无穷小比较的语言后,我们可以建立一个“坐标系”,将一个简单的无穷小作为参照物,进而比较其他复杂无穷小量的阶数,因此我们引入基本无穷小的定义:

Definition:当$x\to x_0$时,称$(x-x_0)$为基本无穷小,若$\alpha (x) \textasciitilde A(x - x_0)^{r}(A \ne 0,r > 0)$则称$\alpha(x)$为$r$阶无穷小,并称$ A(x - x_0)^{r}$为$\alpha(x)$的主部,且$\alpha (x) = A(x-x_0)^{r}+o((x-x_0)^r)$

给出几个例子:

exp1:$\alpha(x) = \cfrac{2x}{1+x^2}(x \to 0),\alpha(x) \textasciitilde 2x$

exp2:$\beta(x) = \ln x^2(x\to1),\beta(x) =\ln x^2 = \ln(1+x^2-1)\textasciitilde (x^2-1) = (x+1)(x-1) = 2(x-1)$

无穷小代换

引入无穷小这一概念后,我们求极限的方法又多了一种,记无穷小代换,其数学语言叙述如下:

Theorem:当$x\to x_0$时,$\alpha \textasciitilde \alpha’,\beta \textasciitilde \beta’$,则有$\lim\limits_{x\to x_0}\cfrac{\alpha}{\beta} =\lim\limits_{x\to x_0}\cfrac{\alpha’}{\beta’} $

给出几个例子:

  • 与$x$等价的无穷小(默认$x \to x_0$)

    $x,\sin x,\arcsin x,\tan x,\arctan x,\ln(1+x),e^x-1,n(\sqrt[n]{1+x}-1)$

  • $1-\cos x\textasciitilde \cfrac{1}{2}x^2,x-\sin x \textasciitilde \cfrac{1}{6}x^3$

exp2:计算$\lim\limits_{x\to 0}\cfrac{\sqrt{4+x^2}}{\sqrt{9+x^3}}\cdot \cfrac{\arcsin x}{e^x-1}\cdot \cfrac{\tan x}{\sin x}\cdot \cfrac{\ln (1+x)}{\arctan x}$

Solution:根据等价无穷小代换,有:

$\lim\limits_{x\to 0}\cfrac{\sqrt{4+x^2}}{\sqrt{9+x^3}}\cdot \cfrac{\arcsin x}{e^x-1}\cdot \cfrac{\tan x}{\sin x}\cdot \cfrac{\ln (1+x)}{\arctan x} = \lim\limits_{x\to 0}\cfrac{\sqrt{4+x^2}}{\sqrt{9+x^3}}\cdot \cfrac{x}{x}\cdot \cfrac{x}{x}\cdot \cfrac{x}{x} = \cfrac{2}{3}$

无穷大

给出无穷大的定义:

Definition:若$\lim\limits_{x\to x_0}L(x) = \infty$,则称$L(x)$为$x\to x_0$的无穷大（量）.

对于无穷大量,我们有一个常用的定理:

Theorem:当$x\to x_0$时,$L(x)$为无穷大量,则$\cfrac{1}{L(x)}$为无穷小量.

最后再给出一个例题:

exp1:计算$\lim\limits_{x \rightarrow 0} \cfrac{(1+m x)^n-(1+n x)^m}{x^2} $

Solution:$\lim\limits_{x \rightarrow 0} \cfrac{(1+m x)^n-(1+n x)^m}{x^2} =\lim\limits_{x \rightarrow 0}\frac{\frac{1}{2} n(n-1) m^2 x^2+o\left(x^2\right)-\frac{1}{2} m(m-1)n^2 x^2-o(x^2)}{x^2}=\lim\limits_{x \rightarrow 0} \frac{\frac{1}{2}\left[\left(n(n-1) m^2-m(m-1) n^2\right] x^2+0\left(x^2\right)\right.}{x^2}=\frac{1}{2} m n(n-m) $

等价无穷小代换及应用

等价无穷小代换是求解极限的一种常用的方法,下面我们给出几道例题来细细体验一下这种方法:

exp1:计算$\lim\limits_{x\to 0}(\cfrac{x+\sin x}{x})$

Solution:$I = \lim\limits_{x\to 0}(\cfrac{x+x}{x}) =2$

exp2:计算$\lim\limits_{x\to 0}(\cfrac{x-\sin x}{x^3})$

Solution:$I = \lim\limits_{x\to 0}(\cfrac{\frac{1}{6}x^3}{x^3}) = \cfrac{1}{6}$

针对前两道例题,我们可以给出一个推论:

Theorem:当$x\to x_0$时,若$\alpha \textasciitilde \alpha’,\beta \textasciitilde \beta’,r\textasciitilde r’$,且$\lim\limits_{x\to x_0}\cfrac{\alpha}{r} \ne -1$,则有$\lim\limits_{x\to x_0}\cfrac{\alpha +r}{\beta} = \lim\limits_{x\to x_0}\cfrac{\alpha’ +r’}{\beta’}$

得到这个推论后,我们再来看下列例题:

exp3:计算$\lim\limits_{x\to 0}\cfrac{x\cos x-x+x^3+o(x^3)}{x^3}$

Solution:根据以上推论,$I = \lim\limits_{x\to 0}[\cfrac{x\cos x-x}{x^3}+\cfrac{x^3+o(x^3)}{x^3}] = \cfrac{1}{2}$

exp4:计算$\lim\limits_{x \to x_0}\cfrac{\tan^3(2x)}{x^4}(1-\cfrac{x}{e^x-1})$

Solution:$I = 8\lim\limits_{x\to0}[\cfrac{e^x-x-1}{x(e^x-1)}] = 8\lim\limits_{x\to 0}\cfrac{\frac{1}{2}x^2+o(x^2)}{x^2}=4$

exp5:计算$\lim\limits_{x \rightarrow 0} \cfrac{3 \sin x+x^2 \cos \frac{1}{x}}{(1+\cos x) \ln (1+x)}$

Solution:$I = \lim\limits_{x \rightarrow 0}\left(\cfrac{3 \sin x+x \cos x}{2 x}\right)=\cfrac{3}{2}$

exp6:计算$\lim\limits_{x \rightarrow 0}(\cos x)^{\frac{1}{\ln \left(1+x^2\right)}}$

Solution:$I = \lim\limits_{x \rightarrow 0}\left\{(1+\cos x-1)^{\frac{1}{\cos x-1}}\right\}^{\frac{\cos x-1}{\ln \left(1+x^2\right)}}=e^{-\frac{1}{2}}$

exp7:计算$\lim\limits_{x \rightarrow 0} \cfrac{\sqrt{1+x}+\sqrt{1-x}-2}{x^2}$

Solution:

$\begin{align}I &= \lim\limits_{x \rightarrow 0} \cfrac{\sqrt{1+x}-1+\sqrt{1-x}-1}{x^2} \\ & =\lim\limits_{x \rightarrow 0} \cfrac{\frac{x}{\sqrt{1+x}+1}-\frac{x}{\sqrt{1-x}+1}}{x^2} \\ &= \cfrac{1}{4}\lim _{x \rightarrow 0} \cfrac{\sqrt{1-x}-\sqrt{1+x}}{x} \\ &=\cfrac{1}{4} \lim\limits_{x \rightarrow 0} \cfrac{-2 x}{(\sqrt{1-x}+\sqrt{1+x}) x} \\ &= -\cfrac{1}{4}\end{align}$

exp8:计算$\lim\limits_{x \rightarrow 0} \cfrac{[\sin x-\sin (\sin x)] \sin x}{x^4}$

Solution: $I = \lim\limits_{x \rightarrow 0}\cfrac{\sin(x) -\sin (\sin x)}{x^3} = \lim\limits_{x \rightarrow 0}\cfrac{x-\sin x}{x^3}= \cfrac{1}{6}$

连续与间断

连续

在讨论极限过后,函数的研究方法又多了一种.此前我们在中学阶段接触到的函数均为连续函数,但是我们并未用严谨的语言对连续做一个定义,下面给出连续的极限定义:

Definition:设$y = f(x)$在$x = x_0$及其附近有定义,对于$\forall \varepsilon > 0,\exists \delta > 0$,当$x \in N(x_0,\delta)$,有$|f(x)-f(x_0) < \varepsilon|$,即$\lim\limits_{x \to x_0}f(x) = f(x_0)$,则称$y=f(x)$在$x = x_0$处连续.

以上是双侧连续的定义,同理,我们可以给出单侧极限的定义,定义如下:

Definition:

  • $\lim\limits_{x\to x_0^{+}}f(x) = f(x_0)$,则称函数$f(x)$在$x=x_0$右连续.
  • $\lim\limits_{x\to x_0^{-}}f(x) = f(x_0)$,则称函数$f(x)$在$x=x_0$左连续.

定义单侧连续后,我们立马可以得到一个重要的推论:

Theorem:$f(x)$在$x = x_0$处连续$\Leftrightarrow \lim\limits_{x\to x_0^{+}}f(x) = \lim\limits_{x\to x_0^{-}}f(x) = f(x_0)$

此后,我们给出$f(x)$在闭区间$[a,b]$上连续的定义:

Definition$f(x)$在$[a,b]$上连续$\Leftrightarrow f(x)$在$(a,b)$上任意一点连续,且在$x=a,b$处单侧连续.将在$[a,b]$上连续的函数,记作$C[a,b]$

在定义连续后,我们需要讨论函数的连续性,有以下定理:

Theorem:连续函数的和差积商（分母不为0）及复合函数仍为连续函数.

Theorem:严格单调连续函数的及其反函数仍为严格单调连续函数.

Theorem:基本初等函数在其定义域内是连续的

关于该定理的证明,直接利用连续的$\varepsilon-\delta$语言即可,这里不再详细说明.

间断

我们遇到的函数不一定是连续的,其图像往往会在某些点出现“断开”的现象.我们用间断来描述.下面给出间断的严格定义:

Definition:若$\lim\limits_{x \to x_0^+}f(x) = \lim\limits_{x \to x_0^-}f(x)=f(x_0) $不成立,称$f(x)$在$x = x_0$处间断.

第一类间断点

为了更方便地描述间断,我们将函数的间断点分类为第一类间断点与第二类间断点.

下面给出第一类间断点的定义:

Definition:函数$f(x)$在$x = x_0$处间断,但是$\lim\limits_{x \to x_0^+}f(x),\lim\limits_{x \to x_0^-}f(x)$均存在.

  • 进一步分类讨论,当$\lim\limits_{x \to x_0^+}f(x)=\lim\limits_{x \to x_0^-}f(x) $时,称该间断点为可去间断点.如函数$f(x) = \cfrac{\sin x}{x}$在$x_0 = 0 $处.
  • 当$\lim\limits_{x \to x_0^+}f(x)\ne\lim\limits_{x \to x_0^-}f(x) $时,称该间断点为跳跃间断点.如函数$f(x) = [x]$在$x_0=1,2,3…$处.

第二类间断点

再给出第二类间断点的定义:

Definition:函数$f(x)$在$x = x_0$处间断,但是$\lim\limits_{x \to x_0^+}f(x),\lim\limits_{x \to x_0^-}f(x)$至少有一个不存在.

  • 进一步讨论,考虑函数$f(x) = \cfrac{1}{x}$,其在$x_0=0$处的间断点被称为无穷间断点.
  • 再考虑函数$f(x) = \sin \cfrac{1}{x}$,其在$x_0 = 0$处的间断点被称为震荡间断点.

下面给出几个例题吧:

exp1:考虑函数$f(x) = \cfrac{x}{\sin x}$,求其间断点及类型

Solution:显然,$x_0 = n\pi (n \in Z)$是其间断点

当$x_0 = 0$时,$\lim\limits_{x \to 0^+}f(x)=\lim\limits_{x \to 0^-}f(x)= 1$,故$x_0 = 0$是可去间断点.

当$x_0 = n\pi(n \ne 0)$时,$f(x) \to \infty$,故$x_0 = n\pi $是第二类间断点（无穷间断点）

exp2:考虑函数$f(x) = \begin{cases}(1+x)^{1/x} ,x\ne 0 \\ 1,x =0 \end{cases}$,求其间断点及类型.

Solution:易知,$x_0 = 0$是其间断点.

当$x_0 = 0$时,$\lim\limits_{x \to 0^+}f(x)=\lim\limits_{x \to 0^-}f(x) = e \ne f(0)$,故$x_0 = 0$是可去间断点

exp3:考虑函数$f(x) = \arctan \cfrac{1}{x}$,求其间断点及类型.

Solution:易知$x_0 = 0$是其间断点

当$x_0=0$时,$\lim\limits_{x \to 0^+}f(x) = \cfrac{\pi}{2},\lim\limits_{x \to 0^-}f(x) = -\cfrac{\pi}{2}$,故$x_0$是跳跃间断点

exp4:考虑函数$f(x)= \lim\limits_{n\to \infty}\cfrac{nx}{nx^2+1}$,求其间断点及类型.

Solution:分情况讨论:当$x = 0$时,$f(x) = 0$,当$x \ne 0 $时,$f(x) = \cfrac{1}{x}$.

即$f(x) = \begin{cases}\cfrac{1}{x},x\ne 0 \\ 0,x= 0\end{cases}$,易知$x_0 = 0$是其间断点.

当$x\to 0$时,$f(x) \to \infty$,故$x_0 = 0$ 是第二类间断点（无穷间断点）.

exp5:考虑函数$f(x) = \lim\limits_{n \to \infty}\cfrac{x+e^{xt}}{1+e^{xt}}$,求其间断点及类型.

Solution:分情况讨论:

当$x > 0$时,$f(x) = 1$.

当$x = 0$时,$f(x) = \cfrac{1}{2}$.

当$x< 0$时,$f(x) = x$

易知$x_0 = 0$是其间断点,$\lim\limits_{x \to 0^{+}}f(x) = 1,\lim\limits_{x \to 0^-}f(x) = 0$,故$f(x)$是跳跃间断点

Note:在求间断点类型的题目中,我们经常遇到函数的极限表示式,遇到这种表达形式,需要分类讨论.

有限闭区间连续函数的性质

最后我们来讨论一下$f(x) \in C[a,b]$的性质吧！

有界性定理

考虑一个函数$f(x)\in C[a,b]$,其满足有界性定理:

Theorem(有界性定理):$f(x)$在$[a,b]$上一定有界.

Note1:考虑函数$f(x) = \cfrac{1}{x},x\in(0,1)$,易知$f(x)$在$(0,1)$连续,但是$f(x)$在$(0,1)$无界,故有界性定理只在闭区间上成立.

Note2:考虑函数$f(x) = \begin{cases}\cfrac{1}{x},x\in(0,1] \\ 1,x = 0\end{cases}$,易知$f(x)$$(0,1]$上连续但是无界,故有界性定理要求函数在闭区间上连续.

最值定理

考虑一个函数$f(x) \in C[a,b]$,其满足最值定理:

Theorem(最值定理):$\exists x_1,x_2 \in [a,b]$使得$m \leq f(x_1) \leq f(x_2) \leq M$

Note1:考虑函数$f(x) = \cfrac{1}{1+e^{1/x}},x \in [-1,1]$,易知$f(x)$在$[-1,1]$上无法取到最值,故最值定理要求函数在闭区间上连续.

Note2:考虑函数$f(x) = x,x\in (-1,1)$,易知$f(x)$在$(-1,1)$上无法取到最值,故最值定理只在闭区间上成立.

介值定理

考虑一个函数$f(x) \in C[a,b]$,其满足介值定理:

Theorem(介值定理):若$\mu$是介于$f(a),f(b)$之间的任何值,则$\exists \xi \in [a,b]$使得$f(\xi ) = \mu$

特别地,连续函数$f(x)$在$[a,b]$上的最大值,最小值分别设为$M,m$,则对于$\forall \mu \in [m,M],\exists \xi \in [a,b]$使得$f(\xi) = \mu$.

零点存在性定理

考虑函数$f(x) \in C[a,b]$,其满足零点存在性定理:

Theorem:若函数$f(x)$满足$f(a)\cdot f(b) < 0$,则$\exists \xi \in (a,b)$,使得$f(\xi) = 0$

利用零点存在性定理,我们往往可以判断一个连续函数的零点个数及其范围,给出几个例子:

exp1:求方程$x^3+4x^2-3x-1=0$解的个数.

Solution:记函数$f(x) =x^3+4x^2-3x-1$,

易知$f(-10) = -5 < 0,f(-1) = 5 > 0 ,f(0) = -1,f(1) = 1> 0$

故根据零点存在性定理,方程有两个解

exp2:求方程$4-x = 2^x$在$[1,2]$的解的个数

Solution:记$f(x) = 4-x-2^x,x\in[1,2]$

易知$f(1) = 1,f(2)=-2$,且函数在$[1,2]$上单调递减,则有一个实根

例题

关于本部分,最后给出几个例题:

exp1:$\lim\limits_{x\to -1}\cfrac{ax^2-x-3}{x+1} = b$,求$a,b$的值.

Solution:$I = -2a-1 = b$

由极限的存在性可知$a = 2,b = -5$

exp2:已知$\lim\limits_{n\to \infty}(3n-\sqrt{an^2+bn+1}) = 2$,求$a,b$的值.

Solution:$I = \lim\limits_{n\to \infty}\cfrac{(9-a)n^2-bn-1}{3n+\sqrt{an^2+bn+1}} = 2$

由极限的存在性可知$a= 9,b=-12$

exp3:已知函数$f(x) = \lim\limits_{n\to\infty}\cfrac{x^{2n-1}+ax^2+bx}{x^{2n}+1}$在$R$上连续,求$a,b$的值.

Solution:分情况讨论,可知函数$f(x)$的表达形式为:

$f(x)=\begin{cases} \cfrac{1}{x},x<-1 \\\cfrac{1}{2}(a-b-1),x = -1\\ ax^2+bx,-11\end{cases}$

根据连续的定义,我们可以求得$a=0,b=1$
