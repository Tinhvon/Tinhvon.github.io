极限与连续

间断

间断的定义

我们遇到的函数不一定是连续的,其图像往往会在某些点出现"“断开”的现象.我们用间断来描述.下面给出间断的严格定义:

Definition:若${x x_0^+}f(x) = {x x_0^-}f(x)=f(x_0) \(不成立,称\)f(x)\(在\)x = x_0$处间断.

第一类间断点

为了更方便地描述间断,我们将函数的间断点分类为第一类间断点与第二类间断点.

下面给出第一类间断点的定义:

Definition:函数\(f(x)\)在\(x = x_0\)处间断,但是\(\lim\limits_{x \to x_0^+}f(x),\lim\limits_{x \to x_0^-}f(x)\)均存在.

  • 进一步分类讨论,当${x x_0^+}f(x)={x x_0^-}f(x) \(时,称该间断点为**可去间断点**.如函数\)f(x) = \(在\)x_0 = 0 $处.
  • 当${x x_0^+}f(x){x x_0^-}f(x) \(时,称该间断点为**跳跃间断点**.如函数\)f(x) = [x]\(在\)x_0=1,2,3...$处.

第二类间断点

再给出第二类间断点的定义:

Definition:函数\(f(x)\)在\(x = x_0\)处间断,但是\(\lim\limits_{x \to x_0^+}f(x),\lim\limits_{x \to x_0^-}f(x)\)至少有一个不存在.

  • 进一步讨论,考虑函数\(f(x) = \cfrac{1}{x}\),其在\(x_0=0\)处的间断点被称为无穷间断点.
  • 再考虑函数\(f(x) = \sin \cfrac{1}{x}\),其在\(x_0 = 0\)处的间断点被称为震荡间断点.

下面给出几个例题吧:

Example 1:考虑函数\(f(x) = \cfrac{x}{\sin x}\),求其间断点及类型

Solution 显然,\(x_0 = n\pi (n \in Z)\)是其间断点

当\(x_0 = 0\)时,\(\lim\limits_{x \to 0^+}f(x)=\lim\limits_{x \to 0^-}f(x)= 1\),故\(x_0 = 0\)是可去间断点.

当\(x_0 = n\pi(n \ne 0)\)时,\(f(x) \to \infty\),故$x_0 = n$是第二类间断点(无穷间断点)

Example 2:考虑函数\(f(x) = \begin{cases}(1+x)^{1/x} ,x\ne 0 \\ 1,x =0 \end{cases}\),求其间断点及类型.

Solution 易知,\(x_0 = 0\)是其间断点.

当\(x_0 = 0\)时,\(\lim\limits_{x \to 0^+}f(x)=\lim\limits_{x \to 0^-}f(x) = e \ne f(0)\),故\(x_0 = 0\)是可去间断点

Example 3:考虑函数\(f(x) = \arctan \cfrac{1}{x}\),求其间断点及类型.

Solution 易知\(x_0 = 0\)是其间断点

当\(x_0=0\)时,\(\lim\limits_{x \to 0^+}f(x) = \cfrac{\pi}{2},\lim\limits_{x \to 0^-}f(x) = -\cfrac{\pi}{2}\),故\(x_0\)是跳跃间断点

Example 4:考虑函数\(f(x)= \lim\limits_{n\to \infty}\cfrac{nx}{nx^2+1}\),求其间断点及类型.

Solution 分情况讨论:当\(x = 0\)时,\(f(x) = 0\),当$x \(时,\)f(x) = $.

即\(f(x) = \begin{cases}\cfrac{1}{x},x\ne 0 \\ 0,x= 0\end{cases}\),易知\(x_0 = 0\)是其间断点.

当\(x\to 0\)时,\(f(x) \to \infty\),故\(x_0 = 0\) 是第二类间断点(无穷间断点).

Example 5:考虑函数\(f(x) = \lim\limits_{n \to \infty}\cfrac{x+e^{xt}}{1+e^{xt}}\),求其间断点及类型.

Solution 分情况讨论:

当\(x > 0\)时,\(f(x) = 1\).

当\(x = 0\)时,\(f(x) = \cfrac{1}{2}\).

当\(x< 0\)时,\(f(x) = x\)

易知\(x_0 = 0\)是其间断点,\(\lim\limits_{x \to 0^{+}}f(x) = 1,\lim\limits_{x \to 0^-}f(x) = 0\),故\(f(x)\)是跳跃间断点

Note 1 :在求间断点类型的题目中,我们经常遇到函数的极限表示式,遇到这种表达形式,需要分类讨论.
