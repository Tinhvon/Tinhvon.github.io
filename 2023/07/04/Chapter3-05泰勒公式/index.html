Chapter3-05泰勒公式

微分中值定理与导数的应用

泰勒公式

在开始前

我们在学习微分的过程中,曾使用过函数的线性近似公式,即: \[ f(x) = f(x_0)+f'(\xi)(x-x_0) \\ f(x) = f(x_0)+f'(\xi)(x-x_0)+o(x-x_0) \] 受其启发,对于函数\(f(x)\),我们能否构造一个简单函数(如多项式\(p(x)\)),使得\(|f(x)-p(x)|\)可以为任意小,如果这样的多项式\(p(x)\)存在,则:

  • \(f(x)\)应当满足何条件?
  • \(p(x)\)如何构造?
  • \(p(x)\)是否唯一？

讨论具有以下性质的\(f(x)\):

\(f(x)\)在\(x=x_0\)及其附近有\(n+1\)阶导数,寻找关于\((x-x_0)\)的多项式\(P_n(x) = a_n(x-x_0)^n+...+a_1(x-x_0)+a_0\),用\(P_n(x)\)来逼近\(f(x)\)使得\(f(x)-P_n(x) = o((x-x_0)^n)\).

其中\(P_n(x)\)满足条件\(P_n^{(k)}(x) = f^{(k)}(x),k=0,1,2,...,n\)由此可得: \[ a_k = \cfrac{1}{k!}f^{(k)}(x),k=0,1,2,...,n \\ P_n(x) = f(x_0)+f'(x_0)(x-x_0)+...+\cfrac{f^{(n)}(x_0)}{n!}(x-x_0)^n \]

泰勒中值定理(Taylor公式)

下面我们给出泰勒中值定理.其表述如下:

Taylor Theorem:如果\(y=f(x)\)在\((a,b)\)处有\(n+1\)阶导数,\(x_0 \in (a,b)\),则对于\(\forall x \in (a,b),f(x) = f(x_0)+f'(x_0)(x-x_0)+...+\cfrac{1}{n!}f^{(n)}(x_0)(x-x_0)^n + R_n(x) = P_n(x)+R_n(x)\).其中\(R_n(x) = \cfrac{1}{(n+1)!}f^{(n+1)}(\xi)(x-x_0)^{n+1}\),其中\(\xi\)介于\(x,x_0\)之间

证明如下:

Proof 记\(R_n(x)=f(x)-P_n(x),R_n^{(k)}(x_0)=0,k=0,1,...n\)

由柯西中值定理可知:

\(\cfrac{R_n(x)}{(x-x_0)^{n+1}} = \cfrac{R_n(x)-R_n(x_1)}{(x-x_0)^{n+1}} = \cfrac{R_n'(\xi_1)}{(n+1)(\xi_1-x_0)^n}=...=\cfrac{1}{(n+1)!}\cfrac{R^{(n)}(\xi_n-R^{n}(x_0))}{\xi_0-x_0}\)

根据拉格朗日中值定理,原式$ = R^{n+1}(),(x,x_0)$

而\(R^{(n+1)(x)} = f^{(n+1)}(x),R_n(x) = \cfrac{1}{(n+1)!}f^{(n+1)}(\xi)(x-x_0)^{n+1}\)

Note 1:一般称\(f(x) = f(x_0)+f'(x_0)(x-x_0)+...+\cfrac{1}{n!}f^{(n)}(x_0)(x-x_0)^n + R_n(x) = P_n(x)+R_n(x)\)为\(n\)阶\(Taylor\)公式,\(P_n(x)\)称为\(n\)次\(Taylor\)多项式,由\(|f^{(n+1)}(x)|\leq M,|R_n(x)|\leq \cfrac{M}{(n+1)!}|(x-x_0)^{n+1}| \Rightarrow \cfrac{R_n(x)}{(x-x_0)^n} \to 0,x \to x_0\).

Note 2:我们一般称\(R_n(x) = \cfrac{1}{(n+1)!}f^{(n+1}(\xi)(x-x_0)^{n+1}\)为Lagrange型余项,\(R_n(x)= o(x-x_0)^n\)为Peano型余项.当\(x_0=0\)时,我们称Taylor公式为Maclaurin公式.

Note 3:常见函数的Maclaurin公式如下: \[ e^x = 1+x+\cfrac{x^2}{2!}+...+\cfrac{x^n}{n!}+\cfrac{e^{\theta x}}{(n+1)!}x^{n+1},R_n(x) = o(x^n),x\in R \\ \sin x =x - \cfrac{x^3}{3!}+...+(-1)^{n-1}\cfrac{x^{2n-1}}{(2n-1)!}+(-1)^n\cfrac{\cos \theta x}{(2n+1)!}x^{2n+1},x\in R \\ \cos x = 1 - \cfrac{x^2}{2}+...+(-1)^n\cfrac{x^{2n}}{(2n)!}+(-1)^{n+1}\cfrac{\cos \theta x}{(2n+2)!}x^{2n+2},R_n(x) = o(x^{2n}) \\ \ln(1+x) = x - \cfrac{x^2}{2}+...+(-1)^{n}\cfrac{x^{n+1}}{(n+1)(1+\theta x)^{n+1}},R_n(x) = o(x^n),x\in (-1,1] \\ (1+x)^{\alpha} = 1+\alpha x+...+\cfrac{\alpha (\alpha -1)...(\alpha -n+1)}{n!}x^n + \cfrac{\alpha (\alpha -1)...(\alpha -n)}{(n+1)!(1+\theta x)^{n+1}}x^{n+1},R_n(x) = o(x^n) \\ \cfrac{1}{1-x} = 1+x+x^2+...+x^n+o(x^n),x\in (-1,1) \\ \cfrac{1}{1+x} = 1-x+x^2-...+(-1)^nx^n+o(x^n),x\in (-1,1) \] 下面给出一些Taylor展开的例题:

Example 1:\(f(x) = \sin (x^2)\),写出其Maclaurin展开式.

Solution 将\(\sin x\)的Maclaurin展开式中的\(x\)全部用\(x^2\)代替即可

Example 2:\(f(x) = xe^{x+1}\),写出其Maclaurin展开式

Solution \(f(x) = exe^x = ex(\sum\limits_{k=0}^n\cfrac{x^k}{k!})+o(x^n+1) = e\sum\limits_{k=0}^n\cfrac{x^{k+1}}{k!}+o(x^n+1),x\in R\)

应用I 求解极限

泰勒公式在求解较复杂的极限时有很大的用途,下面给出几个例题:

Example 1:计算\(\lim\limits_{x\to0}\cfrac{\cos x-e^{-x^2/2}+x^4}{x^4}\)

Solution

\(\begin{align}I &= \lim\limits_{x\to0}\cfrac{[1-\frac{x^2}{2!}+\frac{x^4}{4!}+o(x^4)]-[1-\frac{x^2}{2}+\frac{x^4}{8}+o(x^4)]+x^4}{x^4} \\ &= \lim\limits_{x\to0} \cfrac{\frac{11}{12}x^4+o(x^4)}{x^4} \\ &= \cfrac{11}{12}\end{align}\)

Example 2:计算\(\lim\limits_{x\to0}\cfrac{e^x\sin x-x(1+x)}{x^3}\)

Solution

\(\begin{align}I &= \lim\limits_{x\to0}\cfrac{[1+x+\frac{x^2}{2}+o(x^3)][x-\frac{x^3}{3!}+o(x^3)]-x-x^2}{x^3} \\ &= \lim\limits_{x\to0}\cfrac{\frac{1}{3}x^3+o(x^3)}{x^3} \\ &= \cfrac{1}{3}\end{align}\)

Example 3:计算\(\lim\limits_{x \to 0}\cfrac{e^x - \sum\limits_{n=0}^{99}\frac{x^n}{n!}}{x^{100}}\)

Solution

\(I=\lim\limits_{x \rightarrow 0} \cfrac{\sum\limits_{n=0}^{99} \frac{x^n}{n !}+\frac{x^{100}}{100 !}+o\left(x^{100}\right)-\sum\limits_{n=0}^{99} \frac{x^{n}}{n!}}{x^{100}} = \cfrac{1}{100!}\)
