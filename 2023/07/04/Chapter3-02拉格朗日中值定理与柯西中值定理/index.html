Chapter3-02拉格朗日中值定理与柯西中值定理

微分中值定理与导数应用

微分中值定理(II)

我们在上一篇博文中介绍了费马原理,然后用费马原理成功导出了第一个微分中值定理——罗尔定理.在这篇文章中,我们会导出剩余的微分中值定理,以及给出一些相关的习题.

拉格朗日中值定理

我们直接给出拉格朗日中值定理的表述:

Lagrange Theorem:设\(y=f(x)\)满足在\([a,b]\)上连续,在\((a,b)\)上可导,则\(\exists \xi \in (a,b)\),使得\(f'(\xi) = \cfrac{f(b)-f(a)}{b-a}\).

证明如下:

Proof 构造函数\(L(x) = f(x)-f(a)-\cfrac{f(b)-f(a)}{b-a}(x-a)\).

显然我们有\(L(a) = L(b)\).

根据罗尔定理:\(\exist \xi \in (a,b)\),使得\(L'(\xi) = 0\)

即:\(f'(\xi) = \cfrac{f(b)-f(a)}{b-a}\).

Note 1:如图可知,拉格朗日中值定理是罗尔定理的推广.

Note 2:拉格朗日中值定理的另一种写法为:\(f(x+\Delta x)-f(x) = f'(x+\theta \Delta x)\Delta x(0<\theta <1)\).

Note 3:根据拉格朗日中值定理,我们可以得到一个推论:若\(|f'(x)|\leq L,x\in(a,b)\),则有\(|f(x_2)-f(x_1)| \leq L|x_2-x_1|\).

柯西中值定理

最后我们来给出最后一个微分中值定理——柯西中值定理,其表述如下:

Caucht Theorem:设函数\(F(x),G(x)\)满足在\([a,b]\)上连续,在\((a,b)\)上可导,且\(\forall x \in (a,b),G'(x)\ne 0\).则\(\exists \xi \in (a,b)\),使得\(\cfrac{F(b)-F(a)}{G(b)-G(a)} = \cfrac{F'(\xi)}{G'(\xi)}\).

证明如下:

Proof 构造函数\(C(x) = F(x)-F(a)-\cfrac{F(b)-F(a)}{G(b)-G(a)}[G(x)-G(a)]\)

显然我们有\(C(a)=C(b)\)

根据罗尔定理:\(\exists \xi \in (a,b)\),使得\(C'(\xi) = 0\)

即\(\cfrac{F(b)-F(a)}{G(b)-G(a)} = \cfrac{F'(\xi)}{G'(\xi)}\)

Note 1:显然,柯西中值定理是拉格朗日中值定理的推广.

Example 设\(x_1x_2>0\),求证:\(\exists \xi \in (x_1,x_2)\),使得\(x_1e^{x_2}-x_2e^{x_1}=(1-\xi)e^{\xi}(x_1-x_2)\)

Proof 即证明\(\cfrac{\frac{e^{x_2}}{x_2}-\frac{e^{x_1}}{x_1}}{\frac{1}{x_2}-\frac{1}{x_1}} = (1-\xi)e^{\xi}\)

设函数\(F(x) = \cfrac{e^x}{x},G(x) = \cfrac{1}{x}\),利用柯西中值定理,可证.

达布定理

最后我们介绍一个函数论中很重要的定理——达布定理,其表述如下:

Darboux Theorem:设\(y=f(x)\)在\([a,b]\)可导,则\(f'(x)\)至少有一次取得介于\(f'_+(a),f'_-(b)\)之间的任何值.

证明如下:

Proof

  • 当\(f'_+(a)f'_-(b)<0\)时,即证明\(\exists \xi \in (a,b),f'(\xi) = 0\).不妨假设\(f'_+(a)<0,f'_-(b)>0\),此时\(f(x)\)在\((a,b)\)内部取到最小值.根据柯西中值定理可证.
  • 设\(c\)是介于\(f'_+(a),f'_-(b)\)之间的某值,构造\(D(x)=f(x)-cx\),则有\(D'_+(a)D'_-(b)<0\),此时由之前的推论可证.
