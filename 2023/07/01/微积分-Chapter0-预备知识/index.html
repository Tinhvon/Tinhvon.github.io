微积分-Chapter0-预备知识

预备知识

集合

集合

首先给出集合的定义

定义:具有某种特定性质的元素的汇总所成的集体.其中,构成集合的对象称为元素.一般,用大写字母表示集合,如\(N\),\(R\),\(Q\)等；而用小写字母表示元素,如\(x\),\(y\)等.常记为: \[ A = \{ x | x \in A\} \quad or\quad A = \{x|x的特性\} \]

一般表示集合的方法有:

列举法:\(A = \{ 1,3,5,7,...,2021\}\)

特性表示法:\(A = \{x| x > 1986\}\)

了解集合的定义后,便可以讨论集合之间的关系与集合之间的运算.

首先讨论集合之间的关系,定义如下:

若\(\forall x \in A \Rightarrow x \in B\),则称\(A\)是\(B\)的子集,记为\(A \subseteq B\).

若\(A \subseteq B,\exists x \in B,x \notin A\),则称\(A\)是\(B\)的真子集.

若\(A \subseteq B,B \subseteq A\),称两个集合相等,记为\(A=B\)

再讨论集合之间的运算,定义如下:

并集:\(A \cup B = \{x \in A \quad or \quad x \in B\}\)

交集:\(A \cap B = \{x \in A \quad and \quad x \in B\}\)

差集:\(A - B = \{x \in A \quad and \quad x \notin B\}\)

补集:定义一个非空集合\(X\),\(A \subseteq X\),则\(A^{c} = X - A\)是\(A\)的补集.

乘积:考虑集合$A = { x | x A},B = { y | yA} \(,则\)A B ={ (x,y) | x A ,y B} $

区间

定义集合后,我们可以用区间来表述数轴上的一个范围,现有两个实数\(a,b\),不妨假设\(a < b\),则定义如下:

\((a,b) = \{x|a<x<b\}\),称该区间为开区间

\([a,b] = \{x|a\leq x \leq b\}\),称该区间为闭区间

\((a,b] =\{x|a < x \leq b \}\)与\([a,b) =\{x|a \leq x < b \}\)统称为半开半闭区间.

\((a,+∞) = \{x|x > a\}\)

\((-∞,b] = \{x|x \leq b\}\)

\((-∞,+∞) = \{x|-∞ < x < +∞ \} = R\)

邻域

在函数论中,我们经常遇到邻域的概念,假设\(x_0 \in R,\delta > 0\),则:

\(x_0\)的\(\delta\)邻域定义为:\(N(x_0,\delta) = \{x| \quad|x-x_0|<\delta \}\)

\(x_0\)的\(\delta\)去心邻域定义为:\(N_0(x_0,\delta) =\{x| 0<|x-x_0|<\delta \}\)

下面给出常用邻域的例子:

右邻域:\(N^{+}(x_0,\delta) = \{x| 0\leq x-x_0<\delta \}\)

去心右邻域:\(N^{+}_{0}(x_0,\delta) = \{x|0< x-x_0<\delta \}\)

左邻域:\(N^{-}(x_0,\delta) = \{x| -\delta < x-x_0\leq 0 \}\)

去心左邻域:\(N^{-}_{0}(x_0,\delta) = \{x| -\delta < x-x_0\leq 0 \}\)

\(+∞\)邻域:即\((a,+∞)\)

\(-∞\)邻域:即\((-∞,b)\)

\(∞\)邻域:即\((-∞,b) \cup (a,+∞)\)

常用不等式

高中阶段学过的常见不等式如下(取等条件略):

绝对值不等式:\(|a_1+a_2+...+a_n| \leq |a_1|+|a_2|+... + |a_n|\)

Bernoulli 不等式: 对于\(\forall x > -1\),\((1+x)^{n}\geq 1+nx\).

平均值不等式:\(x_i>0,i=1,2,...n\),有\(\sqrt[n]{x_1x_2...x_n}\leq \cfrac{x_1+x_2+...+x_n}{n}\)

实数理论初步

该部分严格证明需要学习《数学分析》,普通微积分对此不作要求,了解即可.

实数有以下性质:

1.实数对四则运算是封闭的,即实数通过四则运算仍为实数.

2.实数是有序集合,数学语言表述即为: \[ \forall a,b\in R \Rightarrow a<b或a=b或a>b \] 3.实数具有稠密性,即任意两个实数之间一定存在实数.

4.实数与数轴上的点一一对应.

确界

现引入确界概念

实数集\(E\)的最小上界定义为上确界,用\(\sup E\)表示.

实数集\(E\)的最大下界定义为下确界,用\(\inf E\)表示.

实数理论中很重要的一个定理是确界存在原理,其表述为

设集合\(S\)为非空实数集,若\(S\)有上界,则\(S\)必有上确界；若\(S\)有下界,则\(S\)必有下确界.

映射与函数

映射

下面引入映射的定义:

设\(A,B\)为两个非空集合,若存在对应法则\(f\),使得\(\forall x\in A\),存在唯一的\(y \in B\)与之对应,则称\(f\)为$A B $的一个映射,记为: \[ f:A\to B或f:x\to y或y=f(x) \] 称\(y\)为映射下的像,称\(A\)为\(f\)的定义域,\(f(A)\)为\(f\)的值域.

由定义可知,映射的三要素分别为定义域、值域、对应法则.

函数

定义映射后,我们讨论一种特殊的映射—函数.其定义为:

考虑映射\(f\),若其定义域与值域均为实数集,则称\(f\)是一个(实)函数.

一元函数的表示为:\(f:R\to R\)

n元函数的表示为:\(f:R^{n} \to R\)

下面给出几个特殊的函数例子

符号函数: \[ \mathrm{sgn} (x) = \begin{cases} -1 ,x<0 \\ 0, x= 0 \\ 1,x >0 \end{cases} \] 狄利克雷函数: \[ D(x) = \begin{cases} 1,x\in Q \\ 0,x\notin Q \end{cases} \] 黎曼函数: \[ R(x) = \begin{cases} \cfrac{1}{q},x = \cfrac{p}{q},q>p>0 \\ 0,x=0,1or无理数 \end{cases} \]

我们同样可以定义函数的运算:

四则运算

与实数一样,也可以定义函数的加减乘除,其中记\(\alpha f(x)+\beta g(x)\)是函数\(f(x),g(x)\)的线性组合

复合运算

设函数\(f(x),g(x)\),称\(f[g(x)]\)为\(f(x),g(x)\)的复合函数,其中定义域为 \[ D = \{ x|x \in D(g)andg(x)\in D(f)\} \]

反函数运算

考虑函数\(y=f(x),x\in D,\forall x_1 \ne x_2 \Rightarrow f(x_1)\ne f(x_2)\),此时可以推断映射\(f\)是单射,满射,记为\(f:D\to J\),此时我们可定义映射\(g:D \to J\),称\(g\)是\(f\)的反函数,记为\(g=f^{-1}\)

函数有以下特性:

1.有界性:若函数\(f(x)\)满足\(\forall x \in D,|f(x)|\leq M\),则称函数有界.

2.单调性:对于\(\forall x_1 \ne x_2\),\((x_2-x_1)[f(x_2)-f(x_1)] \leq (\geq)0\),则称函数单调递减(增),若\(\leq\)改为\(<\)(\(\geq\)改为>),则称为严格单调递减(增).

3.奇偶性:若函数定义域关于原点对称,且\(f(x)=f(-x)\),则称函数为偶函数；若\(f(x)=-f(-x)\),则称函数为奇函数

4.周期性:\(\exists T>0\),若\(f(x+T)=f(x)\),则称\(y=f(x)\)为以\(T\)为周期的函数,满足条件的$T的最小值成为函数的最小正周期.

最后给出基本初等函数:

常函数:即\(f(x) = C\),\(C\)为常数.

幂函数:即\(f(x) = x^{\alpha} = e^{\alpha \ln x}\)

指数函数:即\(f(x) = a^{x}(a>0,a\ne1)\),常用的指数函数为\(f(x) = e^{x}\)

对数函数:即\(f(x) = \log_{a}(x)(a>0,a\ne1)\),若底数为\(e\),则函数为\(f(x) = \ln x\)

三角函数:常见的三角函数为\(\sin x,\cos x,\tan x ,\cot x,\sec x,\csc x\)

反三角函数:表示方法为\(\arcsin x,\arccos x,\arctan x\)

双曲函数:双曲正弦函数\(\sinh x = \frac{e^x-e^{-x}}{2}\),双曲余弦为\(\cosh x = \frac{e^x+e^{-x}}{2}\),双曲正切为:\(\tanh x = \frac{e^x-e^{-x}}{e^x+e^{-x}}\)

初等函数可以由基本函数表示.
