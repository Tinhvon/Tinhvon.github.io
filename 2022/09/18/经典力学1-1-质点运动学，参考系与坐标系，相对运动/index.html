经典力学-Chapter1-矢量力学

Chapter1 矢量力学

1.1质点运动学，参考系与坐标系，相对运动

前言：作者本人已经大二啦，开始了《理论力学》与《数学物理方法》的学习。本系列文章是作者平时的课堂笔记+一些自己的思考，希望能和大家共同交流。

什么是运动学，运动学主要研究系统在时空中的运动，并使用几何手段来描述物体的运动。在矢量力学中，时间和空间是分离的，空间可以使用Euclid几何来描述。同时，矢量力学中的空间是均匀的也是各向同性的，而时间是均匀的。在经典力学中，时空的均匀性决定了一系列守恒律！

而为了描述物体的运动，我们需要引入自己的一套“语言体系”。首先为了简化对物理问题的研究，我们引入了质点这模型。其严格定义为：**没有大小形状，只有质量的点。**显然，质点模型是一个高度理想化理想模型，当我们研究一个物理问题时，物体的形状，体积对研究几乎甚至没有影响时，我们可以“放弃”对这些次要因素的研究，而是用一个只有质量的点来等效这个物体，从而对其进行研究。

但是仅有质点是仅仅不够的，我们需要更多的“语言”来描述运动！由此产生了基本物理量。考虑到基本物理量都是用于描述质点在某一瞬时的运动状态，故其有瞬时性；此外，在不同的参考系下，我们观测到的基本物理量往往是不同的，这被称为相对性；在运动学中，物理量的变化往往有大小和方向两个方向，所以其有矢量性。描述运动的基本物理量如下：
位矢 r⃗\vec{r}r:在参考系已经选定的情况下，某个确定的点指向质点所在位置的矢量，它用来描述质点位置随时间变化的规律。在运动学中，r⃗=r⃗(t)\vec{r}= \vec{r}(t)r=r(t)被称为运动方程式，其对时间 $ t $ 单值可微。而 r⃗\vec{r}r在空间划过的曲线，通常被称为轨迹。
位移 Δr⃗\Delta \vec{r}Δr :在给定位矢r⃗\vec{r}r后，为了描述在某一段时间Δt\Delta tΔt内位矢的变化。我们引入位移这一概念，显然，它是$ t,\Delta t $的函数，表示为

r⃗(t,Δt)=r⃗(t+Δt)−r⃗(t)\vec{r}(t,\Delta t) = \vec{r}(t+\Delta t) - \vec{r}(t) r(t,Δt)=r(t+Δt)−r(t)

速度 v⃗\vec{v}v :速度通常定义为位移与完成这段位移所需时间的比值，即：

v⃗=dr⃗dt\vec{v}= \cfrac{\mathrm{d}\vec{r} }{\mathrm dt} v=dtdr​

形象地讲，速度常用来描述运动的快慢和方向，同位矢，它也是时间的单值可微函数。
加速度a⃗\vec{a}a :加速度通常定义为速度该变量与完成这些改变所需时间的比值，即:

a⃗=dv⃗dt\vec{a}= \cfrac{\mathrm d\vec{v}}{\mathrm d t} a=dtdv​

同理，加速度通常用来描述速度改变的快慢与，方向。

在基本物理量给出后，我们便需要观测（测量）这些物理量。自然而然，我们引入了参考系这一工具，参考系被称为用来基准的物体或物体组。其定量化结果就是我们所说的坐标系。在坐标系中，矢量可以用分量形式来描述，下介绍几种常见的坐标系：

A⃗=∑i=13Aiei⃗\vec{A} = \sum_{i = 1}^{3}{A_{i}}\vec{e_{i}} A=i=1∑3​Ai​ei​​

Ai=A⃗⋅ei⃗A_{i} = \vec{A}\cdot\vec{e_{i}} Ai​=A⋅ei​​

自然坐标系：当质点被约束在给定轨迹上运动时，常利用此坐标系。其坐标原点为取在轨迹上某瞬间与运动质点重合的点。在该坐标系中，质点的位置通过弧坐标 sss 确定，用单位矢量$ \vec{e_{\tau}}$ 表示，称为切向量。与此同时，主法线的正方向指向运动轨迹的凹侧并且垂直于eτ⃗\vec{e_{\tau}}eτ​​ ，用单位矢量en⃗\vec{e_{n}}en​​ 表示。由切线，主法线，次发现的正方向组成右手螺旋系规定次法线的正方向，该正方向用单位矢量 eb⃗\vec{e_{b}}eb​​表示，根据定义，有：eb⃗=eτ⃗×en⃗\vec{e_{b}} = \vec{e_{\tau}} \times \vec{e_{n}}eb​​=eτ​​×en​​基底选用切向量和法向量来描述。在该坐标系下，运动的基本物理量可表示为：

在上面的讨论中，我们知道物体的同一运动在不同的参考系下，得到的结果也不同。但是客观上，物体的统一运动对应着唯一的物理过程，所以不同的参考系之间的运动学量也存在着对应关系，这便是我们引入相对运动的原因。

运动学中，我们常常会引入静止参考系与运动参考系来进行研究，自然而然，运动的质点相对于运动参考系的运动被称为相对运动，而相对于静止参考系的运动被称为绝对运动，由于运动参考系的牵引，导致质点相对于静止参考系的运动被称为**牵连运动。**在引入这些工具后，我们便能定量地对相对运动进行研究。

考虑任一矢量的时间变化率，不妨考虑两种对时间的微商：①绝对微商 ddt\cfrac{\mathrm d}{\mathrm d t}dtd​ ,相对于静止参考系的时间变化率；②相对微商 d~dt\cfrac{\mathrm {\tilde{d}}}{\mathrm dt}dtd~​,相对于运动参考系的时间变化率。两者之间存在着以下的对应关系：

da⃗dt=d~a⃗dt+ω⃗×a⃗\cfrac{\mathrm d\vec{a}}{\mathrm d t} = \cfrac{\mathrm {\tilde{d}\vec{a}}}{\mathrm dt} + \vec{\omega}\times\vec{a} dtda​=dtd~a​+ω×a

其中ω⃗\vec{\omega}ω 为运动参考系相对于静止参考系的角速度。

现在考虑一个位矢，其在静止参考系下表示为r⃗\vec{r}r ,在运动参考系下表示为 r′⃗\vec{r'}r′ ,显然，两位矢的关系为：

r′⃗=r0⃗+r⃗\vec{r'} = \vec{ r_ {0 } }+\vec{r} r′=r0​​+r

根据速度的定义式，有速度合成定理：

v′⃗=dr0⃗dt+dr⃗dt=v0⃗+d~r⃗dt+ω⃗×r⃗=v0⃗+vr⃗+ω⃗×r⃗\vec{v'} = \cfrac{\mathrm d\vec{r_{0}}}{dt}+\cfrac{\mathrm d\vec{r}}{dt} = \vec{v_{0}}+\cfrac{\mathrm {\tilde{d}}\vec{r}}{dt}+\vec{\omega}\times\vec{r} = \vec{v_{0}}+\vec{v_{r}}+\vec{\omega}\times\vec{r} v′=dtdr0​​​+dtdr​=v0​​+dtd~r​+ω×r=v0​​+vr​​+ω×r

其中相对速度为:

vr⃗=d~r⃗dt\vec{ v_{ r } } = \cfrac{ \mathrm{ \tilde{d}\vec{r } } }{\mathrm dt} vr​​=dtd~r​

牵连速度为:

v0⃗+ω⃗×r⃗\vec{v_{0}}+\vec{\omega}\times\vec{r} v0​​+ω×r

根据加速度的定义式，有加速度合成定理：

a′⃗=dv⃗dt=a0⃗+ar⃗+ω⃗˙×r⃗+ω×(ω⃗×r⃗)+2ω⃗×vr⃗\vec{a'} = \cfrac{\mathrm{d}\vec{v}}{\mathrm{d}t} = \vec{a_{0}} + \vec{a_{r}}+\dot{\vec{\omega}}\times\vec{r}+\omega\times(\vec{\omega}\times\vec{r})+2\vec{\omega}\times\vec{v_{r}} a′=dtdv​=a0​​+ar​​+ω˙×r+ω×(ω×r)+2ω×vr​​

其中科里奥利加速度为:

ar⃗=d~vr⃗dt\vec{a_{r}} = \cfrac{\mathrm{\tilde{d}\vec{v_{r} } } }{\mathrm dt} ar​​=dtd~vr​​​

ac⃗=2ω⃗×vr⃗\vec{a_{c}} = 2\vec{\omega}\times\vec{ v_{r} } ac​​=2ω×vr​​

a0⃗+ω⃗˙×r⃗+ω×(ω⃗×r⃗)\vec{a_{ 0 } }+\dot{ \vec{\omega } }\times\vec{r}+\omega\times( \vec{ \omega }\times\vec{ r }) a0​​+ω˙×r+ω×(ω×r)

在构建了以上的“语言体系”后，我们便可以完整地描述物体的运动了！
