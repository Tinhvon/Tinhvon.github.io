经典力学1.1 质点运动学，参考系与坐标系，相对运动

#Chapter1 矢量力学 ##1.1质点运动学，参考系与坐标系，相对运动 前言：作者本人已经大二啦，开始了《理论力学》与《数学物理方法》的学习。本系列文章是作者平时的课堂笔记+一些自己的思考，希望能和大家共同交流。

什么是运动学，运动学主要研究系统在时空中的运动，并使用几何手段来描述物体的运动。在矢量力学中，时间和空间是分离的，空间可以使用Euclid几何来描述。同时，矢量力学中的空间是均匀的也是各向同性的，而时间是均匀的。在经典力学中，时空的均匀性决定了一系列守恒律！

而为了描述物体的运动，我们需要引入自己的一套“语言体系”。首先为了简化对物理问题的研究，我们引入了质点这模型。其严格定义为：没有大小形状，只有质量的点。显然，质点模型是一个高度理想化理想模型，当我们研究一个物理问题时，物体的形状，体积对研究几乎甚至没有影响时，我们可以“放弃”对这些次要因素的研究，而是用一个只有质量的点来等效这个物体，从而对其进行研究。

但是仅有质点是仅仅不够的，我们需要更多的“语言”来描述运动！由此产生了基本物理量。考虑到基本物理量都是用于描述质点在某一瞬时的运动状态，故其有瞬时性；此外，在不同的参考系下，我们观测到的基本物理量往往是不同的，这被称为相对性；在运动学中，物理量的变化往往有大小和方向两个方向，所以其有矢量性。描述运动的基本物理量如下：
位矢\(\overrightarrow{r}\):在参考系已经选定的情况下，某个确定的点指向质点所在位置的矢量，它用来描述质点位置随时间变化的规律。在运动学中，\[\overrightarrow{r}= \overrightarrow{r}(t)\]被称为运动方程式，其对时间 \(t\) 单值可微。而 \(\overrightarrow{r}\)在空间划过的曲线，通常被称为轨迹。
位移 \(\Delta \overrightarrow{r}\) :在给定位矢\(\overrightarrow{r}\)后，为了描述在某一段时间\(\Delta t\)内位矢的变化。我们引入位移这一概念，显然，它是$ t,t $的函数，表示为： r→(t,Δt)=r→(t+Δt)−r→(t)=Δr→ .
速度 \(\overrightarrow{v}\) :速度通常定义为位移与完成这段位移所需时间的比值，即 \(\overrightarrow{v}= \frac{\mathrm{d}\overrightarrow{r} }{\mathrm dt}\) .形象地讲，速度常用来描述运动的快慢和方向，同位矢，它也是时间的单值可微函数。
加速度\(\overrightarrow{a}\) :加速度通常定义为速度该变量与完成这些改变所需时间的比值，即$= $.同理，加速度通常用来描述速度改变的快慢与方向。

在基本物理量给出后，我们便需要观测（测量）这些物理量。自然而然，我们引入了参考系这一工具，参考系被称为用来基准的物体或物体组。其定量化结果就是我们所说的坐标系。在坐标系中，矢量可以用分量形式来描述，下介绍几种常见的坐标系：

标准正交坐标系：通常也称为笛卡尔坐标系，其基底满足 $ = $ (即我们所说的右手系)。在标准坐标系中，给定矢量\(\overrightarrow{A}\)可表示为：$ \[\begin{aligned} \overrightarrow{A} = \sum_{i = 1}^{3}{A_{i}}\overrightarrow{e_{i}} \end{aligned}\] $,其中 $ \[\begin{aligned} A_{i} = \overrightarrow{A}\cdot\overrightarrow{e_{i}} \end{aligned}\] $ .即： $ \[\begin{aligned} \overrightarrow{A} = \sum_{i = 1}^{3}{\overrightarrow{A}\cdot\overrightarrow{e_{i}}} \overrightarrow{e_{j}} \end{aligned}\]

$

在笛卡尔坐标系中，描述运动的基本物理量可写为： \[ \overrightarrow{r} = x\overrightarrow{e_{i}}+y\overrightarrow{e_{j}}+z\overrightarrow{e_{k}} \\ \overrightarrow{v} = \dot{x}\overrightarrow{e_{i}}+\dot{y}\overrightarrow{e_{j}}+\dot{z}\overrightarrow{e_{k}}\\ \overrightarrow{a} = \ddot{x}\overrightarrow{e_{i}}+\ddot{y}\overrightarrow{e_{j}}+\ddot{z}\overrightarrow{e_{k}} \] 平面极坐标系：其基底与笛卡尔坐标系基底满足变换关系： \(\begin{cases} \overrightarrow{e_{r}} = \cos\theta \overrightarrow{e_{i}}+\sin\theta \overrightarrow{e_{j}}\\ \overrightarrow{e_{\theta}} = -\sin\theta \overrightarrow{e_{i}}+\cos\theta \overrightarrow{e_{j}}\\ \end{cases}\) .显然，在平面极坐标系下，基本物理量可表示成如下形式： \[ \overrightarrow{r} = r\overrightarrow{e_{r}}\\ \overrightarrow{v} = \dot{r}\overrightarrow{e_{r}}+r\dot{\theta}\overrightarrow{e_{\theta}}\\ \overrightarrow{a} = (\ddot{r}-r\dot{\theta}^{2})\overrightarrow{e_{r}}+(2\dot{r}\dot{\theta}+r\ddot{\theta})\overrightarrow{e_{\theta}} \] 数学推导如下：

先考虑证明：\(\dot{\overrightarrow{e_{r}}} = \dot{\theta}\overrightarrow{e_{\theta}},\dot{\overrightarrow{e_{\theta}}} = -\dot{\theta}\overrightarrow{e_{r}}\) ,利用极坐标与直角坐标基底的变换关系可轻易证明。

现考虑位矢：\(\overrightarrow{r} = r\overrightarrow{e_{r}}\) .

根据速度的定义式：

\(\overrightarrow{v} = \frac{\mathrm d\overrightarrow{r}}{dt} = \frac{\mathrm{d}r}{\mathrm dt}\overrightarrow{e_{r}}+r\dot{\overrightarrow{e_{r}}} =\dot{r}\overrightarrow{e_{r}}+r\dot\theta\overrightarrow{e_{\theta}}\).

再根据加速度的定义式：

\(\overrightarrow{a} = \frac{\mathrm d\overrightarrow{v}}{dt} = (\ddot{r}-r\dot{\theta}^{2})\overrightarrow{e_{r}}+(2\dot{r}\dot{\theta}+r\ddot{\theta})\overrightarrow{e_{\theta}}\)

自然坐标系：当质点被约束在给定轨迹上运动时，常利用此坐标系。其坐标原点为取在轨迹上某瞬间与运动质点重合的点。在该坐标系中，质点的位置通过弧坐标 \(s\) 确定，用单位矢量$ $ 表示，称为切向量。与此同时，主法线的正方向指向运动轨迹的凹侧并且垂直于\(\overrightarrow{e_{\tau}}\) ，用单位矢量\(\overrightarrow{e_{n}}\) 表示。由切线，主法线，次发现的正方向组成右手螺旋系规定次法线的正方向，该正方向用单位矢量 \(\overrightarrow{e_{b}}\)表示，根据定义，有：\(\overrightarrow{e_{b}} = \overrightarrow{e_{\tau}} \times \overrightarrow{e_{n}}\)基底选用切向量和法向量来描述。在该坐标系下，运动的基本物理量可表示为： \[ \overrightarrow{r} = \overrightarrow{r}(s),s = s(t)\\ \overrightarrow{v} = \dot{r}\overrightarrow{e_{\tau}}\\ \overrightarrow{a} = \ddot{r}\overrightarrow{e_{\tau}}+\frac{v^{2}}{\rho}\overrightarrow{e_{n}} \] 值得一提的是，这里加速度有两个分量 \(a_{\tau}\) 和 \(a_{n}\) ,分别被称为切向加速度与法向加速度。其中\(a_{n} = \frac{v^{2}}{\rho}\)中的 $ρ \(被称为质点所在位置轨迹的**曲率半径**,它还有一个定义式：\)= ||$ .

在上面的讨论中，我们知道物体的同一运动在不同的参考系下，得到的结果也不同。但是客观上，物体的统一运动对应着唯一的物理过程，所以不同的参考系之间的运动学量也存在着对应关系，这便是我们引入相对运动的原因。

运动学中，我们常常会引入静止参考系与运动参考系来进行研究，自然而然，运动的质点相对于运动参考系的运动被称为相对运动，而相对于静止参考系的运动被称为绝对运动，由于运动参考系的牵引，导致质点相对于静止参考系的运动被称为牵连运动。在引入这些工具后，我们便能定量地对相对运动进行研究。

考虑任一矢量的时间变化率，不妨考虑两种对时间的微商：①绝对微商 \(\frac{\mathrm d}{\mathrm d t}\) ,相对于静止参考系的时间变化率；②相对微商 \(\frac{\mathrm {\tilde{d}}}{\mathrm dt}\),相对于运动参考系的时间变化率。两者之间存在着以下的对应关系：

\(\frac{\mathrm d\overrightarrow{a}}{\mathrm d t} = \frac{\mathrm {\tilde{d}\overrightarrow{a}}}{\mathrm dt} + \overrightarrow{\omega}\times\overrightarrow{a}\),其中\(\overrightarrow{\omega}\) 为运动参考系相对于静止参考系的角速度。

现在考虑一个位矢，其在静止参考系下表示为\(\overrightarrow{r}\) ,在运动参考系下表示为 \(\overrightarrow{r'}\) ,显然，两位矢的关系为：

\(\overrightarrow{r'} = \overrightarrow{r_{0}}+\overrightarrow{r}\)

根据速度的定义式，有速度合成定理：

\(\overrightarrow{v'} = \frac{\mathrm d\overrightarrow{r_{0}}}{dt}+\frac{\mathrm d\overrightarrow{r}}{dt} = \overrightarrow{v_{0}}+\frac{\mathrm {\tilde{d}}\overrightarrow{r}}{dt}+\overrightarrow{\omega}\times\overrightarrow{r} = \overrightarrow{v_{0}}+\overrightarrow{v_{r}}+\overrightarrow{\omega}\times\overrightarrow{r}\) .

其中 \(\overrightarrow{v_{r}} = \frac{\mathrm{\tilde{d}\overrightarrow{r}}}{\mathrm dt}\) 为相对速度， $+ $为牵连速度。

根据加速度的定义式，有加速度合成定理：

$ = = + ++()+2 $

其中\(\overrightarrow{a_{r}} = \frac{\mathrm{\tilde{d}\overrightarrow{v_{r}}}}{\mathrm dt}\)为相对加速度，$ = 2 \(为科里奥利加速度，而\)++()$ 为牵连加速度.

在构建了以上的“语言体系”后，我们便可以完整地描述物体的运动了！
